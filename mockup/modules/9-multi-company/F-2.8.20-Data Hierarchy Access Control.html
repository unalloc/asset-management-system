Data Isolation Rules

Security Model: All data queries automatically include organization scope filters. Company A users cannot access Company B data.
  • All data queries must include organization scope filter
  • Company Code required for all asset-related queries
  • Cross-company access requires explicit inter-company agreements
  • API endpoints enforce scope-based access control
  • UI prevents navigation to unauthorized data
