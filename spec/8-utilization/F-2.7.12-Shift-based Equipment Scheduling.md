### 3.3 Business Logic

**Shift Types:**
- MORNING_SHIFT: Typically 06:00-14:00
- AFTERNOON_SHIFT: Typically 14:00-22:00
- NIGHT_SHIFT: Typically 22:00-06:00

**Decision Tree:**

```
START
  │
  ▼
Load Production Orders
  │
  ▼
Load Equipment Capabilities
  │
  ▼
Load Maintenance Windows
  │
  ▼
Load Changeover Matrix
  │
  ▼
Optimize Schedule
  │
  ├─► Maximize utilization
  ├─► Minimize changeover
  └─► Balance workload
           │
           ▼
Generate Gantt Chart
           │
           ▼
Conflict Detection
           │
  ├─► Conflicts found ─► Apply resolution rules
  │
  └─► No conflicts ─► Finalize Schedule
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.12-001 | Shift break compliance | `breakDuration >= legalMinimum` | Ensure compliant schedule | 1 |
| BR-2.7.12-002 | Changeover minimization | `changeoverTime > minimumChangeover` | Sequence similar jobs together | 2 |
| BR-2.7.12-003 | Maintenance window | `scheduledMaintenance OVERLAPS slot` | Skip slot; reschedule | 1 |
| BR-2.7.12-004 | Overtime threshold | `shiftHours > standardShiftHours` | Require approval | 1 |

**Calculation Formulas:**

```
Equipment Utilization = SUM(ScheduledHours) / (ShiftHours - ChangeoverHours - MaintenanceHours)

Changeover Score = TotalChangeoverTime / NumberOfChangeovers

Load Balance Index = 1 - (MAX(departmentLoad) - MIN(departmentLoad)) / AVG(departmentLoad)
```
