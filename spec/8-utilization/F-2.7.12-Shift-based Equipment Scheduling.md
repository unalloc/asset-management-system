# F-2.7.12 - Shift-based Equipment Scheduling

## Requirement ID
REQ-FAM-8-2.7.12

## Module
Asset Utilization

## Priority
High

## Requirement Description
Provide shift-based equipment scheduling for manufacturing environment, considering production plans, maintenance windows, and changeover time to optimize equipment utilization and production efficiency.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.12-001 | Shift break compliance | `breakDuration >= legalMinimum` | Ensure compliant schedule | 1 |
| BR-2.7.12-002 | Changeover minimization | `changeoverTime > minimumChangeover` | Sequence similar jobs together | 2 |
| BR-2.7.12-003 | Maintenance window | `scheduledMaintenance OVERLAPS slot` | Skip slot; reschedule | 1 |
| BR-2.7.12-004 | Overtime threshold | `shiftHours > standardShiftHours` | Require approval | 1 |

**Shift Types:**
- MORNING_SHIFT: Typically 06:00-14:00
- AFTERNOON_SHIFT: Typically 14:00-22:00
- NIGHT_SHIFT: Typically 22:00-06:00

**Calculation Formulas:**
```
Equipment Utilization = SUM(ScheduledHours) / (ShiftHours - ChangeoverHours - MaintenanceHours)
Changeover Score = TotalChangeoverTime / NumberOfChangeovers
Load Balance Index = 1 - (MAX(departmentLoad) - MIN(departmentLoad)) / AVG(departmentLoad)
```

**Pre-conditions:**

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-2.7.12-001 | Production plan available | `productionPlan.exists = TRUE` | ERR-PRE-2712-001 | "Production plan must be available" |
| PRE-2.7.12-002 | Equipment capability defined | `capabilities.every(e => e.isDefined = TRUE)` | ERR-PRE-2712-002 | "All equipment capabilities must be defined" |
| PRE-2.7.12-003 | Shift calendar defined | `shiftCalendar.every(s => s.isValid = TRUE)` | ERR-PRE-2712-003 | "Shift calendar must be configured" |

**Post-conditions:**

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-2.7.12-001 | Schedule generated | `schedule.id IS NOT NULL` |
| POST-2.7.12-002 | Conflicts resolved | `conflicts.count = 0 OR conflicts.resolved = TRUE` |

**Edge Cases:**

| Edge Case ID | Scenario | Expected Behavior |
|--------------|----------|-------------------|
| EC-2.7.12-001 | Equipment failure during shift | Reschedule affected jobs; notify impacted departments |
| EC-2.7.12-002 | Urgent order insertion | Apply priority rules; reschedule lower priority |
| EC-2.7.12-003 | Overtime request | Require manager approval; track overtime hours |
