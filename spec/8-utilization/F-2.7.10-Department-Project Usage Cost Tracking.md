# F-2.7.10 - Department-Project Usage Cost Tracking

## Requirement ID
REQ-FAM-8-2.7.10

## Module
Asset Utilization

## Priority
High

## Requirement Description
Track asset usage costs for departments, projects, and cost centers, supporting budget comparison, trend analysis, and responsibility reporting.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.10-001 | Variance threshold | `ABS(variance) > varianceThreshold` | Generate alert | 1 |
| BR-2.7.10-002 | Over-budget warning | `actual > budget * 1.1` | Send warning notification | 1 |
| BR-2.7.10-003 | Trend anomaly | `trend.slope > anomalyThreshold` | Flag for investigation | 2 |

**Cost Center Hierarchy:**
DEPARTMENT → SUB-DEPARTMENT → TEAM → PROJECT

**Calculation Formulas:**
```
Variance = ActualCost - BudgetedCost
VariancePercentage = (Variance / BudgetedCost) * 100
CostPerUnit = TotalCost / OutputUnits
TrendGrowthRate = (CurrentPeriod - PreviousPeriod) / PreviousPeriod * 100
```

**Pre-conditions:**

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-2.7.10-001 | Cost center structure defined | `costCenterHierarchy.exists = TRUE` | ERR-PRE-2710-001 | "Cost center hierarchy must be defined" |
| PRE-2.7.10-002 | Allocation completed | `allocation.status = 'POSTED'` | ERR-PRE-2710-002 | "Cost allocation must be completed first" |
| PRE-2.7.10-003 | Budget data available | `budgetData.exists = TRUE` | ERR-PRE-2710-003 | "Budget data must be available" |

**Post-conditions:**

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-2.7.10-001 | Cost report generated | `report.id IS NOT NULL` |
| POST-2.7.10-002 | Variance analysis completed | `varianceAnalysis.completed = TRUE` |

**Edge Cases:**

| Edge Case ID | Scenario | Expected Behavior |
|--------------|----------|-------------------|
| EC-2.7.10-001 | No budget defined | Use zero-based analysis; flag as no-budget |
| EC-2.7.10-002 | Cost center restructured | Maintain historical mapping; re-aggregate |
| EC-2.7.10-003 | Project spans multiple departments | Allocate project costs per actual usage |

## Multi-Company Data Scope
CompanyCode scopes all cost data. Cost center managers view only their cost center. Finance views all within company. Group level sees consolidated only.
