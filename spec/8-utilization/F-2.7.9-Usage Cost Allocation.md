### 1.1 Requirement Description

Allocate asset usage costs to using departments or projects, supporting multiple allocation methods (time proportion, output proportion, fixed rate), achieving cost transparency and accountability.

### 3.3 Business Logic

**Allocation Methods:**
- TIME_PROPORTION: (UsageHours / TotalUsageHours) * TotalCost
- OUTPUT_PROPORTION: (OutputUnits / TotalOutput) * TotalCost
- FIXED_RATE: FixedRatePerHour * UsageHours
- VALUE_PROPORTION: (AssetValue / TotalAssetValue) * TotalCost
- MIXED: Weighted combination of above

**Cost Elements:**
- Depreciation
- Energy/Consumables
- Maintenance
- Space/Occupancy
- Insurance
- Management Fee

**Decision Tree:**

```
START
  │
  ▼
Collect Cost Elements
  │
  ▼
Collect Usage Data
  │
  ▼
Determine Allocation Method per Asset/Department
  │
  ▼
Apply Allocation Formula
  │
  ▼
Generate Allocation Report
  │
  ▼
Process Adjustments (if any)
  │
  ▼
Create Journal Entries (if integrated)
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.9-001 | Method selection | `allocationMethod = 'DEFAULT'` | Use company default method | 1 |
| BR-2.7.9-002 | Minimum allocation | `allocationAmount < minimumThreshold` | Round to minimum or zero | 2 |
| BR-2.7.9-003 | Inter-company allocation | `providingCompany != usingCompany` | Flag for transfer pricing review | 1 |
| BR-2.7.9-004 | Variance check | `variance > varianceThreshold` | Alert for review | 1 |

**Calculation Formulas:**

```
Time-Based Allocation:
  Allocation = (AssetUsageHours / TotalUsageHours) * TotalCost

Output-Based Allocation:
  Allocation = (AssetOutputUnits / TotalOutputUnits) * TotalCost

Fixed-Rate Allocation:
  Allocation = HourlyRate * AssetUsageHours

Mixed Allocation:
  Allocation = w1 * TimeAllocation + w2 * OutputAllocation + w3 * FixedAllocation
  where w1 + w2 + w3 = 1.0
```

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior |
