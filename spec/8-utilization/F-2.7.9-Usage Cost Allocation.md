# F-2.7.9 - Usage Cost Allocation

## Requirement ID
REQ-FAM-8-2.7.9

## Module
Asset Utilization

## Priority
High

## Requirement Description
Allocate asset usage costs to using departments or projects, supporting multiple allocation methods (time proportion, output proportion, fixed rate), achieving cost transparency and accountability.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.9-001 | Method selection | `allocationMethod = 'DEFAULT'` | Use company default method | 1 |
| BR-2.7.9-002 | Minimum allocation | `allocationAmount < minimumThreshold` | Round to minimum or zero | 2 |
| BR-2.7.9-003 | Inter-company allocation | `providingCompany != usingCompany` | Flag for transfer pricing review | 1 |
| BR-2.7.9-004 | Variance check | `variance > varianceThreshold` | Alert for review | 1 |

**Allocation Methods:**
- TIME_PROPORTION: (UsageHours / TotalUsageHours) * TotalCost
- OUTPUT_PROPORTION: (OutputUnits / TotalOutput) * TotalCost
- FIXED_RATE: FixedRatePerHour * UsageHours
- VALUE_PROPORTION: (AssetValue / TotalAssetValue) * TotalCost
- MIXED: Weighted combination of above

**Cost Elements:**
- Depreciation, Energy/Consumables, Maintenance, Space/Occupancy, Insurance, Management Fee

**Calculation Formulas:**
```
Time-Based Allocation = (AssetUsageHours / TotalUsageHours) * TotalCost
Output-Based Allocation = (AssetOutputUnits / TotalOutputUnits) * TotalCost
Fixed-Rate Allocation = HourlyRate * AssetUsageHours
Mixed Allocation = w1 * TimeAllocation + w2 * OutputAllocation + w3 * FixedAllocation
```
