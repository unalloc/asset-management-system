# F-2.7.5 - Usage Contract Management

## Requirement ID
REQ-FAM-8-2.7.5

## Module
Asset Utilization

## Priority
Medium

## Requirement Description
Manage internal asset usage contracts, including inter-department usage agreements, lease contracts, and service level agreements (SLA), supporting contract creation, approval, execution tracking, and renewal management.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.5-001 | IFRS 16 classification | `contract.type = 'LEASE'` AND `asset.value >= 5000` | Flag for IFRS 16 lease accounting | 1 |
| BR-2.7.5-002 | Auto-renewal evaluation | `expiryDate - 90 days` | Generate renewal evaluation | 2 |
| BR-2.7.5-003 | SLA breach detection | `currentPerformance < slaThreshold` | Alert + breach record | 1 |
| BR-2.7.5-004 | Fee calculation | `contract.feeMethod = 'USAGE_BASED'` | Calculate based on usage data | 1 |

**Pre-conditions:**

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-2.7.5-001 | Asset eligible for contract | `asset.isContractEligible = TRUE` | ERR-PRE-275-001 | "Asset must be marked as contract-eligible" |
| PRE-2.7.5-002 | Contract template available | `template EXISTS FOR contractType` | ERR-PRE-275-002 | "Contract template must be available" |
| PRE-2.7.5-003 | Legal approval defined | `legalWorkflow.isConfigured = TRUE` | ERR-PRE-275-003 | "Legal approval workflow must be configured" |

**Post-conditions:**

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-2.7.5-001 | Contract effective | `contract.status = 'EFFECTIVE'` |
| POST-2.7.5-002 | SLA tracking active | `slaMonitors.every(m => m.active = TRUE)` |
| POST-2.7.5-003 | Renewal reminder set | `reminder.scheduledDate IS NOT NULL` |

**Edge Cases:**

| Edge Case ID | Scenario | Expected Behavior |
|--------------|----------|-------------------|
| EC-2.7.5-001 | Contract auto-renewal | 90-day notice required to cancel |
| EC-2.7.5-002 | SLA breach dispute | Escalate to contract review committee |
| EC-2.7.5-003 | Inter-company transfer | Contract assignment requires both party approval |

## Multi-Company Data Scope
Contracts are scoped to participating companies. Both contracting parties have view/modify rights within their scope. Inter-company contract visibility is limited to both parties only. Legal department has audit rights across all contracts.
