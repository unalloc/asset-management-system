### 3.3 Business Logic

**Prediction Ranges:**
- SHORT_TERM: 1-3 months
- MEDIUM_TERM: 3-12 months
- LONG_TERM: 1-3 years

**Decision Tree:**

```
START
  │
  ▼
Load Historical Data + Business Forecast
  │
  ▼
Select Prediction Method
  │
  ├─► Time Series ─► ARIMA, Exponential Smoothing
  ├─► Regression ─► Linear, Polynomial
  └─► Machine Learning ─► Random Forest, Neural Network
           │
           ▼
Generate Base Prediction
           │
           ▼
Apply Business Scenario Adjustments
           │
  ├─► Optimistic scenario
  ├─► Pessimistic scenario
  └─► Baseline scenario
           │
           ▼
Calculate Confidence Intervals
           │
           ▼
Capacity Gap/Surplus Analysis
           │
  ├─► Gap identified ─► Procurement recommendation
  ├─► Surplus identified ─► Disposal/sharing recommendation
  └─► Balanced ─► No action
           │
           ▼
Generate Planning Recommendations
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.15-001 | Prediction horizon | `horizon > maxHorizon` | Limit to maximum | 1 |
| BR-2.7.15-002 | Confidence threshold | `confidence < minimumConfidence` | Include warning | 2 |
| BR-2.7.15-003 | Gap threshold | `capacityGap > gapThreshold` | Trigger procurement review | 1 |
| BR-2.7.15-004 | Model retraining | `modelAge > retrainInterval` | Trigger retraining | 1 |

**Calculation Formulas:**

```
Predicted Utilization = f(historical_data, forecast_factors)

Confidence Interval = Predicted ± (z_value * StandardError)

Capacity Gap = PredictedDemand - CurrentCapacity

Recommendation Score = (Benefit - Cost) / ImplementationTime
```
