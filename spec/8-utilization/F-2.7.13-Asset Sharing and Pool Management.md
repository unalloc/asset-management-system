# F-2.7.13 - Asset Sharing and Pool Management

## Requirement ID
REQ-FAM-8-2.7.13

## Module
Asset Utilization

## Priority
Medium

## Requirement Description
Manage shared asset pool, supporting assets joining pool, intra-pool allocation, usage priority, pool performance analysis to improve asset sharing utilization.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.13-001 | Join eligibility | `utilization < poolUtilizationThreshold` | Flag for pool consideration | 2 |
| BR-2.7.13-002 | Priority override | `priority > currentHolder.priority` | Preempt current allocation | 1 |
| BR-2.7.13-003 | Pool utilization target | `actualUtilization < targetUtilization` | Generate redeployment suggestion | 2 |
| BR-2.7.13-004 | Wait list processing | `asset.returned AND waitList.notEmpty` | Auto-allocate to next in wait list | 1 |

**Pool Types:**
- GENERAL: Laptops, office equipment
- SPECIALIZED: Testing equipment, precision instruments
- REGIONAL: Assets shared across nearby locations

**Allocation Methods:**
- First-Come-First-Served
- Priority-Based
- Reservation-Based

**Calculation Formulas:**
```
Pool Utilization = SUM(AssetUsageHours) / (AssetCount * AvailableHours) * 100
Average Wait Time = SUM(waitTime) / NumberOfAllocations
Pool Efficiency = SUM(satisfiedRequests) / SUM(totalRequests) * 100
```
