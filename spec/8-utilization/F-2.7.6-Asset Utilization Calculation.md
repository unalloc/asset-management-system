# F-2.7.6 - Asset Utilization Calculation

## Requirement ID
REQ-FAM-8-2.7.6

## Module
Asset Utilization

## Priority
High

## Requirement Description
Calculate and analyze asset utilization, including time utilization, capacity utilization, value utilization, supporting multiple calculation methods and benchmark comparison.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-2.7.6-001 | Min data threshold | `dataPoints < minimumThreshold` | Flag as estimated; log uncertainty | 2 |
| BR-2.7.6-002 | Outlier detection | `utilization > 100% OR < 0%` | Validate data; flag anomaly | 1 |
| BR-2.7.6-003 | Benchmark comparison | `utilization < benchmark * 0.8` | Generate low utilization alert | 1 |
| BR-2.7.6-004 | Seasonal adjustment | `seasonalFactor IS DEFINED` | Apply seasonal adjustment | 2 |

**Utilization Types:**
- TIME_UTILIZATION = ActualUseHours / AvailableHours
- CAPACITY_UTILIZATION = ActualOutput / DesignCapacity
- VALUE_UTILIZATION = GeneratedValue / AssetValue

**Calculation Formulas:**
```
Time Utilization = (Total Operating Hours / Total Available Hours) * 100
Capacity Utilization = (Actual Output / Design Capacity) * 100
Value Utilization = (Net Book Value of Output / Asset Book Value) * 100
Overall Utilization = w1 * TimeUtil + w2 * CapacityUtil + w3 * ValueUtil
where w1 + w2 + w3 = 1.0
```

**Calculation Methods:**
- SENSOR: Based on IoT sensor data (automatic)
- MANUAL: Based on user-reported data
- HYBRID: Combination of sensor and manual
