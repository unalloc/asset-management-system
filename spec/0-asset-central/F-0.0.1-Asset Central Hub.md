# F-0.0.1 - Asset Central Hub

**Requirement ID:** F-0.0.1  
**Module:** Asset Central (0)  
**Priority:** High

---

## Requirement Description

The system shall provide a centralized Asset Central Hub that serves as the primary entry point for all asset management operations. This hub shall enable users to search, filter, and view statistics for all assets across the organization, providing a unified view that connects all asset-related modules and functionality.

---

## Business Rules

1. Asset Central Hub shall be accessible from the main navigation as the primary entry point for asset operations
2. All assets within the user's scope (Company → Division → Plant → Department → Cost Center) shall be visible
3. Search functionality shall support partial matches on asset ID, name, serial number, and location
4. Filters shall be combinable and persist during the session
5. Asset list shall support sorting by any column in ascending/descending order
6. Pagination shall handle large datasets efficiently (minimum 10,000 assets)
7. User shall be redirected to Asset Detail page when clicking on any asset row

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Search Query | No | Free-text search across asset ID, name, serial, location |
| Filter: Asset Type | No | Filter by asset type category |
| Filter: Status | No | Filter by asset status (Active, Idle, Maintenance, Retired) |
| Filter: Plant | No | Filter by plant location |
| Filter: Department | No | Filter by department |
| Sort Column | No | Column to sort by (default: Asset ID) |
| Sort Direction | No | Ascending or Descending |
| Page Number | No | Page number for pagination |
| Page Size | No | Number of records per page (default: 20) |

---

## Output Data

| Data | Description |
|------|-------------|
| Dashboard Statistics | Total assets, active count, idle count, total book value |
| Filtered Asset List | Table of assets matching search/filter criteria |
| Pagination Info | Current page, total pages, total records |
| Quick Action Links | Links to Dashboard, Reports, Bulk Actions |

---

## Dashboard Statistics

| Statistic | Description |
|-----------|-------------|
| Total Assets | Count of all assets in user's scope |
| Active | Count of assets with status "Active" |
| Idle/Maintenance | Count of assets with status "Idle" or "Under Maintenance" |
| Total Book Value | Sum of current book value for all visible assets |

---

## Asset List Columns

| Column | Description |
|--------|-------------|
| Checkbox | Selection for bulk operations |
| Asset ID | Unique asset identifier |
| Asset Name | Name of the asset |
| Type | Asset type category |
| Status | Current status (Active, Idle, Maintenance, Retired) |
| Location | Plant and physical location |
| Department | Owning department |
| Book Value | Current book value |
| Actions | Quick action buttons (View, Edit, More) |

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full access to all assets across all companies |
| Company Admin | Full access to all assets within own company |
| Division Manager | View assets within assigned division |
| Plant Manager | View assets within assigned plant |
| Department Manager | View assets within assigned department |
| Operator | View assets within assigned department (limited fields) |

---

## Multi-Company Data Scope

- **Data Level**: User's organizational scope (Company → Division → Plant → Department → Cost Center)
- **Default View**: All assets within user's lowest accessible organizational level
- **Cross-Company**: System Admin can view assets across all companies
- **Data Isolation**: Users cannot see assets outside their organizational scope

---

## Compliance Requirements

- **ISO 55000**: Asset Central Hub supports the asset management system overview requirements
- **SOX**: Provides audit trail visibility across all managed assets
- **GDPR**: Only displays asset data; no personal employee data shown without authorization
- **US GAAP/IFRS**: Supports asset register visibility for financial reporting
