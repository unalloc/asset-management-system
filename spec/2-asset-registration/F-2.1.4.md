# F-2.1.4 - Barcode/QR Code Generation

## Requirement ID
F-2.1.4

## Function Name
Barcode/QR Code Generation

## Function Description
Automatically generate asset barcode or QR code labels and support printing functionality.

## Business Rules

1. Barcode encoding rules are configurable
2. Support multiple barcode formats (Code128, QR, etc.)
3. Label templates can be customized
4. Print history recorded

## Preconditions

- Asset is registered
- Barcode printer is configured
- User has "Label Printing" permission

## Postconditions

- Asset label generated and associated
- Print task completed

## Input Data

| Field | Required | Description |
|-------|----------|-------------|
| Asset ID or List | Yes | Asset(s) to label |
| Barcode Format | Yes | Code128/QR/DataMatrix |
| Label Template | No | Template selection |
| Print Quantity | No | Number of copies (default 1) |

## Output Data

1. Barcode image file
2. Print task ID
3. Print status

## Validation Rules

1. Asset must exist and be active
2. Barcode format must be supported
3. Print quantity limit: 1-100 copies

## Error Handling

| Error | Message |
|-------|---------|
| Printer not connected | "Printer is not available" |
| Asset not found | "Asset does not exist" |
| Barcode generation failed | "Barcode generation error" |

## Priority
High

## Related User Stories
