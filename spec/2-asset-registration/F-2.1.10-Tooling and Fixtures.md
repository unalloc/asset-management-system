# F-2.1.10 - Tooling and Fixtures

**Requirement ID:** F-2.1.10  
**Module:** Asset Registration & Cataloging (2.1)  
**Priority:** Medium (for manufacturing enterprises)

---

## Requirement Description

Manage tooling assets such as molds, fixtures, jigs, and gauges including usage counters and maintenance tracking. Tooling assets have design life limits and trigger maintenance when usage thresholds are reached.

---

## Business Rules

1. Tooling types include MOLD, FIXTURE, JIG, CUTTING, and GAUGE
2. Each tooling asset has a unique tooling number within the company
3. Design life must be defined as a positive number
4. Usage counters track how much the tool has been used
5. Maintenance is triggered when usage reaches the defined threshold
6. Warnings are generated when usage reaches 90% of design life
7. Tooling can be rehabilitated and returned to service with a new design life

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Tooling Number | Yes | Unique identifier for the tooling |
| Tooling Type | Yes | Type of tooling (MOLD, FIXTURE, JIG, CUTTING, GAUGE) |
| Applicable Product | No | Product this tooling is designed for |
| Design Life | Yes | Expected usage life before replacement |
| Maintenance Threshold | No | Usage point to trigger maintenance |

---

## Output Data

| Data | Description |
|------|-------------|
| Tooling Asset Record | Created tooling with current usage |
| Usage History | Record of all usage updates |
| Maintenance Plan | Generated plan based on threshold |

---

## Validation Rules

1. Tooling type must be one of the valid types
2. Tooling number must be unique within the company
3. Design life must be greater than 0
4. Maintenance threshold must be between 0 and design life
5. Usage counter starts at 0 for new tooling

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full tooling management across all companies |
| Company Admin | Full management within company |
| Plant Manager | Manage tooling within plant |

---

## Multi-Company Data Scope

- **Data Level**: Company level
- **Access**: Tooling scoped to company
- **Isolation**: Each company has isolated tooling registry

---

## Compliance Requirements

- **OSHA**: Tool safety inspection records per 29 CFR 1910
- **GAMP 5**: Tool qualification records for GMP equipment
- **SOX**: Tool asset capitalization and depreciation tracking
