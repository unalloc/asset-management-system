# F-2.1.10 - Tooling and Fixtures

## Requirement ID
F-2.1.10

## Function Name
Tooling and Fixtures Management

## Function Description
Manage tooling assets such as molds, fixtures, and jigs including usage counters.

## Business Rules

1. Tooling lifecycle management
2. Usage count/time tracking
3. Maintenance cycles based on usage
4. Tooling status management

## Preconditions

- Tooling defined as asset type
- User has "Tooling Management" permission

## Postconditions

- Tooling asset registration completed
- Usage counter initialized

## Input Data

| Field | Required | Description |
|-------|----------|-------------|
| Tooling Type | Yes | Mold/Fixture/Jig/etc |
| Tooling Number | Yes | Unique identifier |
| Applicable Product/Part | No | Products this tool is used for |
| Design Life | Yes | Count or time |
| Initial Usage Count | No | Starting point |
| Maintenance Threshold | No | Trigger point for maintenance |

## Output Data

1. Tooling asset ID
2. Usage counter ID
3. Maintenance plan ID

## Validation Rules

1. Tooling number must be unique
2. Design life must be positive
3. Applicable product must be valid

## Error Handling

| Error | Message |
|-------|---------|
| Duplicate number | "Tooling number already exists" |
| Invalid product | "Applicable product does not exist" |
| Life definition error | "Design life must be greater than 0" |

## Priority
Medium (for manufacturing enterprises)

## Related User Stories
US-041
