# F-2.1.5 - RFID Tag Integration

**Requirement ID:** F-2.1.5  
**Module:** Asset Registration & Cataloging (2.1)  
**Priority:** Medium

---

## Requirement Description

Support RFID tag read/write operations and integration for rapid asset identification and tracking. RFID tags enable automated asset identification without line-of-sight, supporting real-time asset location and inventory workflows.

---

## Business Rules

1. RFID reader must be connected before tag operations
2. Each RFID tag can only be assigned to one asset at a time
3. Each asset can have only one active RFID tag assigned
4. Tag status can be ACTIVE, INACTIVE, or LOST
5. Tags follow GS1 Tag Data Standard for encoding
6. Lost tags can be marked and replaced with new tags
7. All tag operations are logged for audit purposes

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Tag UID | Yes | Unique identifier from RFID tag hardware |
| Tag Type | Yes | Technology type (LF, HF, or UHF) |
| Asset ID | Yes | Asset to associate with the tag |
| Write Data | No | Additional data to write to tag memory |

---

## Output Data

| Data | Description |
|------|-------------|
| Tag Association Record | Link between tag and asset |
| Tag History | Complete history of tag operations |
| RFID Operation Log | Audit trail of all RFID operations |

---

## Validation Rules

1. RFID reader must be connected and operational
2. Tag type must be supported (LF, HF, or UHF)
3. Asset must exist and be in active status
4. Tag UID must be in valid format for the tag type
5. Tag must not already be assigned to another asset
6. Write data must fit within tag memory capacity

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full RFID operations across all companies |
| Company Admin | Full RFID management within own company |
| Plant Manager | Initialize and assign tags within plant |
| Department User | Read tags on assigned assets |

---

## Multi-Company Data Scope

- **Data Level**: Company level
- **Access**: RFID tags are scoped to the company that owns them
- **Isolation**: Tags cannot be assigned across companies

---

## Compliance Requirements

- **GDPR**: RFID tracking data justified under legitimate interest for asset management
- **SOX**: All tag operations logged for asset custody audit trail
- **GAMP 5**: Supports pharmaceutical track-and-trace requirements
