# F-2.1.8 - Manufacturing Asset Types

## Requirement ID
F-2.1.8

## Function Name
Manufacturing Asset Types

## Function Description
Support manufacturing-specific asset types such as production lines, CNC machines, injection molding machines, and assembly equipment.

## Business Rules

1. Predefined manufacturing asset type templates
2. Type-specific attribute sets
3. Production-related parameter configuration
4. Equipment capability definitions

## Preconditions

- Manufacturing module is enabled
- User has "Manufacturing Asset Management" permission

## Postconditions

- Manufacturing asset types available
- Related attribute configuration completed

## Input Data

| Field | Required | Description |
|-------|----------|-------------|
| Asset Type Code | Yes | e.g., PROD_LINE, CNC_MACHINE |
| Type Name | Yes | Display name |
| Equipment Category | Yes | Machining/Forming/Assembly/etc |
| Technical Parameter Template | No | Parameter definitions |
| Capability Metric Definitions | No | Capability indicators |

## Output Data

1. Type creation confirmation
2. Type configuration ID
3. Attribute template ID

## Validation Rules

1. Type code conforms to manufacturing standards
2. Technical parameters defined completely
3. Capability metrics are reasonable

## Error Handling

| Error | Message |
|-------|---------|
| Type code conflict | "Manufacturing type code already exists" |
| Parameter definition error | "Technical parameter format error" |

## Priority
High (for manufacturing enterprises)

## Related User Stories
US-041
