# F-2.1.8 - Manufacturing Asset Types

**Requirement ID:** F-2.1.8  
**Module:** Asset Registration & Cataloging (2.1)  
**Priority:** High (for manufacturing enterprises)

---

## Requirement Description

Support manufacturing-specific asset types such as production lines, CNC machines, injection molding machines, and assembly equipment. Manufacturing types define equipment categories and technical parameters specific to production assets.

---

## Business Rules

1. Manufacturing module must be enabled to use this feature
2. Manufacturing type codes must be unique within a company
3. Equipment categories include MACHINING, FORMING, ASSEMBLY, MATERIAL_HANDLING, PACKAGING, QUALITY, and UTILITIES
4. Technical parameters are required for manufacturing types
5. Each type can define capability metrics for OEE tracking
6. Types follow a lifecycle: DRAFT, ACTIVE, DEPRECATED

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Type Code | Yes | Unique identifier for the type |
| Type Name | Yes | Display name for the type |
| Equipment Category | Yes | Manufacturing category |
| Technical Parameters | Yes | JSON structure with equipment specs |
| Capability Metrics | No | Metrics definitions for OEE tracking |

---

## Output Data

| Data | Description |
|------|-------------|
| Manufacturing Type Record | Created type with all attributes |
| Capability Metrics | Metric definitions linked to type |
| Audit Log | Record of type creation/modification |

---

## Validation Rules

1. Manufacturing module must be enabled
2. Type code must be unique within the company
3. Type code must be 1-30 uppercase alphanumeric characters
4. Equipment category must be one of the valid types
5. Technical parameters must be valid JSON
6. Required technical parameters must be defined

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full manufacturing type management |
| Company Admin | Create/modify types within company |
| Division Manager | View types within division |
| Plant Manager | View types within plant |

---

## Multi-Company Data Scope

- **Data Level**: Company level
- **Access**: Types are scoped to the company level
- **Isolation**: Types cannot be shared across companies

---

## Compliance Requirements

- **OSHA**: Machine-specific safety categories per 29 CFR 1910 subparts
- **EPA**: RCRA compliance for manufacturing waste equipment
- **GAMP 5**: GMP-critical equipment classification for pharmaceutical manufacturing
- **SOX/IFRS/US GAAP**: Manufacturing asset valuation and depreciation tracking
