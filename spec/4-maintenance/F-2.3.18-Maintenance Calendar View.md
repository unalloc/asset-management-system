# F-2.3.18 - Maintenance Calendar View

**Requirement ID:** F-2.3.18  
**Module:** Maintenance Management  
**Priority:** Medium

---

## Requirement Description

Provide visual calendar interfaces including Gantt charts and resource calendars that display scheduled maintenance activities, resource availability, and maintenance windows, enabling better planning and coordination of maintenance work. Gantt chart view for maintenance planning, calendar view (day, week, month) for scheduling, resource calendar showing technician availability, asset operating windows and maintenance blackout periods, drag-and-drop schedule adjustments, conflict detection for resource overbooking, dependency visualization between work orders, maintenance window highlighting, and integration with production scheduling.

---

## Business Rules

1. Conflict detected when same resource assigned to overlapping work orders
2. Finish-to-Start dependency: successor cannot start before predecessor completes
3. Maintenance windows cannot overlap critical production periods
4. Drag-drop rescheduling triggers conflict detection
5. Resource overbooking flagged when utilization exceeds capacity
6. Dependency violations prevent invalid scheduling

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| View Type | Yes | Gantt, Day, Week, or Month |
| Start Date | Yes | Beginning of view range |
| End Date | Yes | End of view range |
| Resource Filter | No | Specific resources to display |
| Asset Filter | No | Specific assets to display |
| Status Filter | No | Work order statuses to include |

---

## Output Data

| Data | Description |
|------|-------------|
| Gantt Chart Data | Work orders with timelines and dependencies |
| Resource Utilization | Hours scheduled vs available |
| Conflict Report | Overlapping assignments |

---

## Validation Rules

1. Date range must be valid and not exceed 1 year
2. No resource assignment conflicts
3. Maintenance windows cannot conflict with critical production

---

## Access Control

| Role | Permission |
|------|------------|
| Maintenance Manager | Full calendar access |
| Plant Supervisor | View and modify maintenance windows |
| Scheduler | Drag-drop rescheduling |

---

## Multi-Company Data Scope

- **Data Level**: Plant level for windows, company level for views
- **Access**: Calendar viewing requires at least Department access
- **Isolation**: Company-level data isolation for all views |

---

- GAMP 5 scheduling controls for validated equipment
