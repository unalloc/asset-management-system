### 1.1 Requirement Description

Manage maintenance contracts with external service providers, tracking contract execution and expenses. Includes contract basic information management (number, period, amount), service scope and SLA terms, payment plan and actual payment records, contract changes and renewal management, vendor performance evaluation, and contract expiration warnings.

## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Contract number must be unique | ContractNumber NOT EXISTS IN Contract WHERE CompanyCode = Session.CompanyCode | SCM-001 | "Contract number already exists in company" |
| PRE-002 | End date must be after start date | EndDate > StartDate | SCM-002 | "Contract end date must be after start date" |
| PRE-003 | Contract amount must be positive | ContractAmount > 0 | SCM-003 | "Contract amount must be positive" |
| PRE-004 | Vendor must exist and be active | Vendor.VendorID = VendorID AND Vendor.Active = TRUE | SCM-004 | "Vendor does not exist or is inactive" |
| PRE-005 | Payment plan total must equal contract amount | SUM(PaymentPlan.Amount) = ContractAmount | SCM-005 | "Payment plan installments must sum to contract amount" |

### 3.2 Post-conditions

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Contract created with calculated expiration | ContractID generated, ExpirationDate calculated |
| POST-002 | Notification scheduled for expiration warning | NotificationLog entry created |

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
[Create Contract]
  │
  ▼
[Validate Vendor]
  │
  ▼
[Validate Payment Plan]
  │
  ▼
[Calculate SLA Compliance]
  │
  ▼
[Schedule Expiration Notification]
  │
  ▼
   END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Auto Expiration Check | Contract created/modified | Set ExpirationDate = EndDate - AdvanceNoticeDays | 1 |
| BR-002 | Payment Plan Validation | Payment plan sum | SUM(Amounts) = ContractAmount | 1 |
| BR-003 | Vendor Performance Score | On service execution | Calculate based on SLA compliance | 2 |
| BR-004 | Renewal Recommendation | 90 days before expiration | Flag for review | 2 |
