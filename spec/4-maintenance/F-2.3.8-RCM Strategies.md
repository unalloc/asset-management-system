# F-2.3.8 - RCM Strategies

**Requirement ID:** F-2.3.8  
**Module:** Maintenance Management  
**Priority:** Medium

---

## Requirement Description

Implement reliability-centered maintenance strategies, optimizing maintenance plans based on risk analysis. Includes equipment function and failure mode analysis, failure consequence assessment (safety, environmental, production, cost), maintenance task type selection (preventive, predictive, corrective), maintenance interval optimization, RCM analysis report and review, and strategy effectiveness tracking and improvement.

---

## Business Rules

1. Risk Priority Number (RPN) calculated as Severity x Occurrence x Detection
2. High risk (RPN >= 200) requires proactive or predictive maintenance strategy
3. Medium risk (RPN 50-199) requires preventive maintenance
4. Low risk (RPN < 50) allows run-to-failure corrective maintenance
5. Maintenance intervals optimized based on MTBF data
6. Strategy effectiveness tracked and compared to predictions

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Asset | Yes | Equipment being analyzed |
| Function Description | Yes | What the equipment is supposed to do |
| Failure Modes | Yes | Potential ways the equipment can fail |
| Severity Rating | Yes | Impact severity (1-10) |
| Occurrence Rating | Yes | Failure likelihood (1-10) |
| Detection Rating | Yes | Ability to detect failure (1-10) |
| Consequence Assessment | Yes | Safety, environmental, production, cost impacts |

---

## Output Data

| Data | Description |
|------|-------------|
| RCM Analysis Report | Complete analysis with recommendations |
| Recommended Strategy | Selected maintenance approach |
| Optimized Intervals | Recommended maintenance frequency |

---

## Validation Rules

1. Equipment function must be documented
2. At least one failure mode must be documented from analysis
3. All consequence categories must be assessed
4. Maintenance options must have valid cost and interval parameters

---

## Access Control

| Role | Permission |
|------|------------|
| Reliability Engineer | Create and manage RCM analyses |
| Engineering Manager | Review and approve strategies |
| Maintenance Manager | View within plant |

---

## Multi-Company Data Scope

- **Data Level**: Company level
- **Access**: RCM analysis requires engineering or reliability specialist role
- **Isolation**: Cross-plant analysis requires Company Admin |

---

## Compliance Requirements

- ISO 55000 risk-based maintenance approach
- OSHA safety consequence assessment requirements
- EPA environmental consequence assessment
- GAMP 5 critical equipment RCM for validated systems
