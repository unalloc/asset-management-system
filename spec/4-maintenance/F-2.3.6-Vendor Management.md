### 1.1 Requirement Description

Manage maintenance service vendor information, qualifications, performance, and business relationships. Includes vendor basic information management, qualification certificates and validity period tracking, service classification and capability assessment, performance scoring and evaluation system, contract history and cooperation records, and vendor risk assessment.

## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Vendor name must be unique per company | VendorName NOT EXISTS IN Vendor WHERE CompanyCode = Session.CompanyCode | VM-001 | "Vendor name already exists in company" |
| PRE-002 | Certificate validity must be reasonable | CertValidTo > CertValidFrom AND CertValidTo > CURRENT_DATE | VM-002 | "Certificate validity dates are invalid" |
| PRE-003 | Contact information must be complete | Email IS NOT NULL OR Phone IS NOT NULL | VM-003 | "At least one contact method required" |
| PRE-004 | Score must be within valid range | Score >= 1 AND Score <= 5 | VM-004 | "Score must be between 1 and 5" |

### 3.2 Post-conditions

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Vendor profile created | VendorID generated |
| POST-002 | Qualification alerts scheduled | Notification for expiring certifications |

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
[Create/Update Vendor]
  │
  ▼
[Validate Unique Name]
  │
  ▼
[Check Certification Status]
  │
  ├─► Expired ──► Flag as ineligible for new contracts
  │
  ├─► Expiring Soon ──► Schedule renewal alert
  │
  ▼
[Calculate Performance Score]
  │
  ▼
   END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Auto Qualification Check | Vendor created/modified | Set QualificationStatus based on cert dates | 1 |
| BR-002 | Performance Score Calculation | On contract completion | Score = (Quality + Timeliness + Cost) / 3 | 2 |
| BR-003 | Risk Assessment | Annual review | Calculate risk score from financial + safety | 1 |
