# F-2.3.5 - Spare Parts Inventory Linkage

**Requirement ID:** F-2.3.5  
**Module:** Maintenance Management  
**Priority:** Medium

---

## Requirement Description

Link maintenance activities with spare parts inventory system, tracking spare parts usage and inventory changes. Maintenance work orders can link required spare parts, spare parts requisition and approval process, actual usage recording and inventory deduction, spare parts usage cost charged to maintenance cost, spare parts inventory alerts (below safety stock), and spare parts usage trend analysis.

---

## Business Rules

1. Parts requisition approved automatically reserves inventory
2. Parts cost automatically allocated to work order
3. Returned parts restore available inventory
4. Safety stock alerts generated when inventory falls below threshold
5. FIFO (First In, First Out) allocation for multiple lots
6. Actual usage cannot exceed requisitioned quantity
7. Serial number tracking for parts that require it

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Work Order | Yes | Work order requesting parts |
| Part | Yes | Spare part being requested |
| Requested Quantity | Yes | Number of parts needed |
| Unit Cost | Yes | Cost per part |
| Status | Yes | Requisition status |
| Requested By | Yes | Person requesting parts |
| Serial Number | No | Serial for tracked parts |
| Lot Number | No | Lot number for lot-tracked parts |

---

## Output Data

| Data | Description |
|------|-------------|
| Parts Reservation | Inventory reserved for work order |
| Cost Allocation | Parts cost added to work order |
| Inventory Transaction | Record of inventory change |

---

## Validation Rules

1. Spare part must exist and have available quantity
2. Requested quantity cannot exceed available inventory
3. Actual usage cannot exceed requisitioned quantity
4. Return quantity cannot exceed unused quantity
5. Serial number must be valid and unused (for tracked parts)
6. Work order must be in Pending, Assigned, or In Progress status

---

## Access Control

| Role | Permission |
|------|------------|
| Technician | Request parts for assigned work orders |
| Plant Supervisor | Approve parts requests |
| Inventory Clerk | Manage inventory adjustments |
| Company Admin | Cross-plant parts transfers |

---

## Multi-Company Data Scope

- **Data Level**: Plant level for inventory operations
- **Access**: Parts linking restricted to assigned technician or supervisor
- **Isolation**: Plant-level inventory is isolated within company |

---

## Compliance Requirements

- SOX parts cost tracking for proper cost center allocation
- EPA RCRA hazardous materials parts tracking
- WEEE Directive electronic spare parts end-of-life
- GAMP 5 parts qualification tracking for regulated equipment
