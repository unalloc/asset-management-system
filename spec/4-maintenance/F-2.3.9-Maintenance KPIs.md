# F-2.3.9 - Maintenance KPIs

**Requirement ID:** F-2.3.9  
**Module:** Maintenance Management  
**Priority:** Medium

---

## Requirement Description

Track and maintain key performance indicators to evaluate maintenance efficiency and effectiveness. Includes standard KPI definitions (MTBF, MTTR, maintenance cost as % of replacement value), automatic data collection and calculation, multi-dimensional analysis (by equipment, department, time), industry benchmarking and target setting, KPI trend analysis and alerting, and improvement measure tracking.

---

## Business Rules

1. KPIs automatically calculated from maintenance and work order data
2. MTBF = Total Operating Time / Number of Failures
3. MTTR = Total Repair Time / Number of Repairs
4. PM Compliance = (Completed PM Work Orders / Scheduled PM Work Orders) x 100
5. Schedule Compliance = (Work Orders Completed On Time / Total Work Orders) x 100
6. Backlog Hours = Sum of all open work order estimated hours
7. Targets can be set per KPI with alert when missed

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Asset | No | Specific asset for KPI calculation |
| Department | No | Department for KPI calculation |
| KPI Code | Yes | Type of KPI to calculate |
| Period Start | Yes | Start of calculation period |
| Period End | Yes | End of calculation period |

---

## Output Data

| Data | Description |
|------|-------------|
| KPI Record | Calculated KPI value with metadata |
| KPI Report | Multi-dimensional analysis report |
| KPI Alert | Notification when target missed |

---

## Validation Rules

1. All required fields for KPI calculation must be available
2. Time range must be valid and not exceed 36 months
3. KPI calculation formula must be defined for the KPI type

---

## Access Control

| Role | Permission |
|------|------------|
| Maintenance Manager | View and set targets |
| Plant Supervisor | View plant KPIs |
| Division Manager | View division KPIs |
| Company Admin | Full access |

---

## Multi-Company Data Scope

- **Data Level**: Company level with rollup to division and plant
- **Access**: Full access at company level, progressively restricted below
- **Isolation**: Company Code filters all KPI data |

---

## Compliance Requirements

- SOX financial KPIs for accurate cost reporting
- OSHA safety KPI tracking
- GAMP 5 equipment effectiveness KPIs for validated systems
