# F-2.2.9 - OEE Calculation

## Requirement ID
F-2.2.9

## Function Name
OEE Calculation

## Function Description
Calculate equipment OEE (Overall Equipment Effectiveness) and analyze overall equipment efficiency.

## Business Rules

1. OEE = Availability × Performance × Quality
2. Support real-time calculation and historical analysis
3. Industry benchmark comparison
4. Improvement opportunity identification

## Input Data

| Field | Required | Description |
|-------|----------|-------------|
| Planned Production Time | Yes | Total available time |
| Actual Runtime | Yes | Time running |
| Theoretical Cycle Time | Yes | Target cycle |
| Actual Output | Yes | Units produced |
| Good Product Quantity | Yes | Defect-free units |

## Output Data

1. OEE value (%)
2. Component values (Availability, Performance, Quality)
3. Loss analysis
4. Improvement suggestions

## Validation Rules

1. Input data must be complete
2. Calculation logic must be correct
3. Result range must be reasonable (0-100%)

## Priority
High (for manufacturing enterprises)
