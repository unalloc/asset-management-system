## Overview

Asset tracking provides visibility into where assets are, what state they are in, and how they are being used. This module ensures:

- Real-time location awareness with geofencing capabilities
- Complete movement history and audit trails
- Integration with IoT for condition monitoring
- Manufacturing-specific metrics like OEE and downtime tracking
