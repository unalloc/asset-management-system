## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-IPT-001 | Source and target plants valid | `SourcePlant.CompanyCode = TargetPlant.CompanyCode OR SourcePlant.DivisionCode = TargetPlant.DivisionCode` | IPT-001 | "Plants must be in same company or division" |
| PRE-IPT-002 | Plants are different | `SourcePlant != TargetPlant` | IPT-002 | "Source and target plants must be different" |
| PRE-IPT-003 | Assets transferable | `Asset.Status = 'Active' AND Asset.IsUnderTransfer = FALSE` | IPT-003 | "Asset is not transferable" |
| PRE-IPT-004 | Approval chain defined | `ApprovalChain.Exists(SourcePlant, TargetPlant) = TRUE` | IPT-004 | "Approval chain not defined for this transfer type" |
| PRE-IPT-005 | Transportation arranged | `Transportation.Mode IS NOT NULL` | IPT-005 | "Transportation mode must be specified" |

### 3.2 Post-conditions

| Post-condition ID | Description | Success Criteria | Verification Method |
|-------------------|-------------|-------------------|---------------------|
| POST-IPT-001 | Transfer ticket created | Unique InterPlantTransferID | Database query |
| POST-IPT-002 | Multi-level approvals recorded | ApprovalChain.Completed = TRUE | Approval query |
| POST-IPT-003 | Transportation tracked | TrackingNumber assigned | Logistics query |
| POST-IPT-004 | Cost allocated | CostAllocation.Distributed = TRUE | Cost query |
| POST-IPT-005 | Asset transferred | Asset.PlantCode = TargetPlant | Asset query |

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Inter-Plant Transfer Request
  │
  ▼
Validate Pre-conditions ────► FAIL ────► Reject with error
  │
  ▼ PASS
Determine Approval Route
  │
  ├──► Same Division ────► Division Manager ────► Plant Managers (both) ────► END
  │
  └──► Different Division ────► Division Managers (both) ────► Company Controller ────► END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-IPT-001 | Cost allocation | Any inter-plant transfer | Calculate and distribute costs | 1 |
| BR-IPT-002 | Transportation booking | Transfer approved | Book transportation | 2 |
| BR-IPT-003 | ETA notification | Transportation arranged | Notify target plant | 2 |
| BR-IPT-004 | Arrival verification | Asset arrives at target | Verify and complete transfer | 1 |
| BR-IPT-005 | Inter-company pricing | Different companies | Apply transfer price | 3 |

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-IPT-001 | Transportation delayed | ETA exceeded | Alert, extend approval window |
| EC-IPT-002 | Asset damaged in transit | Inspection reveals damage | Create incident, insurance claim |
| EC-IPT-003 | Transfer rejected at destination | Target refuses acceptance | Return workflow |
| EC-IPT-004 | Partial transfer | Some assets arrive, others don't | Partial completion, track missing |

### 6.2 Business Rule Validation

| Validation Rule | Description | Condition | Action on Failure | Error Code |

| BRV-IPT-001 | Different plants | SourcePlant != TargetPlant | Reject | IPT-201 |

### 6.3 Error Codes

| Error Code | Severity | Category | Short Description | Long Description |
|------------|----------|----------|-------------------|------------------|
| IPT-001 | Error | Pre-condition | Invalid plant relationship | Plants must be in same company or division |
| IPT-002 | Error | Pre-condition | Same plant | Source and target must be different |
| IPT-003 | Error | Pre-condition | Asset not transferable | Check asset status |
| IPT-004 | Error | Pre-condition | Approval chain missing | Define approval chain first |
| IPT-005 | Error | Pre-condition | Transportation required | Specify transportation mode |
| IPT-101 | Error | Validation | Invalid transport mode | Use TRUCK, RAIL, AIR, SEA, or COURIER |
| IPT-201 | Error | Business Rule | Same plant transfer | Cannot transfer to same plant |
| IPT-202 | Error | Business Rule | Missing hazmat permit | Hazmat assets require transport permit |
