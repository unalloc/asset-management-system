# F-2.5.3 Cross-Factory Consolidated Reporting

## Function ID
F-2.5.3

## Function Name
Multi-Factory Report Consolidation

## Function Description
Generate consolidated reports across multiple factories, supporting drill-down from group level to specific factory details.

## Business Rules
1. Report consolidation rule definition (summary, average, weighted, etc.)
2. Currency conversion and exchange rate handling
3. Data drill-down and dimension switching
4. Time period consistency maintenance
5. Abnormal data marking and handling
6. Report version and snapshot management

## Preconditions
- Participating factories have submitted data
- Consolidation rules are defined
- Exchange rate data is available

## Postconditions
- Consolidated report is generated
- Each factory's contribution is analyzed
- Abnormal data is flagged for review

## Input Data
1. List of participating factories
2. Report consolidation rule configuration
3. Currency conversion exchange rate table
4. Report period (month, quarter, year)
5. Drill-down dimension definition (factory, product line, asset type, etc.)
6. Data abnormal threshold settings
7. Report template and format requirements

## Output Data
1. Consolidated report results
2. Each factory's contribution analysis
3. Currency conversion details
4. Data drill-down path
5. Abnormal data report
6. Report version snapshot
7. Consolidation process audit log

## Validation Rules
1. Participating factory data must be complete
2. Consolidation rules must be consistent and executable
3. Exchange rate data must cover report period
4. Time periods must be aligned
5. Abnormal data must be marked and explained

## Error Handling
1. Incomplete data: Return "Partial factory data incomplete"
2. Rule conflict: Return "Consolidation rules have conflicts"
3. Missing exchange rate: Return "Exchange rate data missing for report period"
4. Time misaligned: Return "Factory data time periods not aligned"
5. Abnormal unhandled: Return "Unhandled abnormal data exists"

## Priority
High (for group management)
