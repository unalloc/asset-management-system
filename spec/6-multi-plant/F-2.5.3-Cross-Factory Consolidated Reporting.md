### 1.1 Requirement Description

Generate consolidated reports across multiple factories, supporting drill-down from group level to specific factory details. This requirement implements the consolidation rules for the multi-company data hierarchy established in F-2.5.1, enabling financial and operational reporting across the organizational hierarchy. Reports must support currency conversion, data validation, and audit trail requirements for SOX and IFRS compliance.

**Calculation Logic (if applicable):**

```
-- Consolidated Value Calculation (Weighted Average)
FUNCTION calculateWeightedAverage(FactoryValues, Weights):
    totalWeight = SUM(Weights)
    weightedSum = SUM(FactoryValues[i] * Weights[i])
    RETURN weightedSum / totalWeight

-- Currency Conversion
FUNCTION convertToReportingCurrency(Amount, SourceCurrency, TargetCurrency, Period):
    IF SourceCurrency = TargetCurrency THEN RETURN Amount
    
    rate = getExchangeRate(SourceCurrency, TargetCurrency, Period.AverageRateDate)
    RETURN Amount * rate
    
-- Outlier Detection (Z-Score)
FUNCTION detectOutliers(Values):
    mean = AVG(Values)
    stdDev = STDDEV(Values)
    
    FOR EACH value IN Values:
        zScore = ABS(value - mean) / stdDev
        IF zScore > 2 THEN
            FLAG value as OUTLIER
        END
    RETURN outliers
```
