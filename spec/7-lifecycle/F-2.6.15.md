# F-2.6.15 Lease Return and Purchase Options

## Function ID
F-2.6.15

## Function Name
Lease Asset Management

## Function Description
Manage lease asset return, renewal, and purchase option execution, including return inspection, purchase price calculation, and ownership transfer.

## Business Rules
1. Lease contract tracking: expiration date, return conditions, purchase options
2. Return process: return inspection, damage assessment, cost calculation
3. Purchase option: fair market value calculation, negotiation, ownership transfer
4. Renewal option: renewal terms, price adjustment, contract update
5. Financial processing: lease termination accounting, purchase capitalization
6. Asset status conversion: lease asset converted to owned asset or returned
7. Supplier coordination: return arrangement, purchase execution
8. Compliance check: lease accounting standards (ASC 842/IFRS 16) compliance

## Preconditions
- Lease contract approaching expiration
- Purchase/return decision made
- Financial resources available (if purchasing)
- Supplier agrees to terms

## Postconditions
- Asset returned or ownership transferred
- Lease contract terminated or renewed
- Financial record updated
- Asset status updated

## Input Data
1. Lease contract ID
2. Asset ID
3. Expiration date
4. Decision (return/purchase/renewal)
5. Purchase price (if applicable)
6. Return inspection result
7. Damage cost (if applicable)
8. New contract terms (if renewal)

## Output Data
1. Lease termination notice
2. Return confirmation or purchase confirmation
3. Ownership transfer documents
4. Financial voucher
5. Asset record update
6. Compliance report

## Validation Rules
1. Purchase price must comply with contract terms or fair market value
2. Return inspection must be mutually acknowledged
3. Financial processing must comply with accounting standards
4. Ownership transfer must be legally effective
5. Decision must be made within contract specified time

## Error Handling
1. Return damage dispute: Negotiate resolution or third-party evaluation
2. Purchase price dispute: Renegotiate or arbitrate
3. Compliance issue: Accounting department involved to adjust
4. Time exceeded: May lose option or incur penalty

## Priority
Medium
