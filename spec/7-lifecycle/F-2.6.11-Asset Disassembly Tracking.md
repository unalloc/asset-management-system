# F-2.6.11 - Asset Disassembly Tracking

**Requirement ID:** F-2.6.11  
**Module:** Asset Lifecycle Management  
**Priority:** High

---

## Requirement Description

Track asset disassembly process, record reusable components, recyclable materials, waste disposal, supporting component inventory management and cost recovery.

---

## Business Rules

1. Asset must be approved for disassembly before process begins
2. Components classified as Reusable, Recyclable, or Waste
3. Hazardous components require certified processor handling
4. Reusable components create inventory records for cost recovery
5. Unknown materials require analysis before disposal
6. WEEE-certified processor required for electronic components
7. RCRA-compliant manifest required for hazardous waste

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| AssetID | Yes | UUID of asset being disassembled |
| FacilityID | Yes | UUID of disassembly facility |
| StartDate | Yes | Disassembly start date/time |
| ComponentType | Yes | Type of component identified |
| MaterialType | Yes | Material composition |
| Classification | Yes | REUSABLE, RECYCLABLE, or WASTE |
| Weight | Yes | Component weight |

---

## Output Data

| Data | Description |
|------|-------------|
| DisassemblySessionID | System-generated UUID |
| ComponentInventoryID | UUID for reusable components added to inventory |
| WasteManifestID | UUID for hazardous waste documentation |
| RecyclingCertificateID | UUID for recycling compliance |

---

## Validation Rules

1. Asset must be approved for disassembly
2. Disassembly facility must be available
3. Certified processor must be contracted for hazardous materials
4. Hazardous component warning triggers certified processor requirement

---

## Access Control

| Role | Permission |
|------|------------|
| SystemAdmin | Full access to disassembly operations |
| AssetManager | Manage disassembly processes |
| EnvironmentalManager | Handle hazardous material tracking |

---

## Multi-Company Data Scope

- **Data Level**: Company/Division/Plant
- **Access**: Disassembly facilities managed at division level
- **Isolation**: CompanyCode filtering on disassembly records

---

## Compliance Requirements

- WEEE: Electronic components recycled through WEEE-certified processors; recycling certificates required; RoHS substance tracking
- EPA/RCRA: Hazardous waste manifest from EPA-certified processor; treatment proof required
- SOX: Component value tracking for gain/loss calculation
- OSHA: Disassembly safety procedures documentation

---
