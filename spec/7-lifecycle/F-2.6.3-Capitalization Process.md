# F-2.6.3 - Capitalization Process

## Function ID
F-2.6.3

## Function Name
CIP to Fixed Asset Conversion

## Function Description
Manage conversion process from construction in progress (CIP) to fixed assets, including cost accumulation, capitalization condition judgment, and accounting voucher generation.

## Business Rules
1. Support automatic and manual capitalization trigger
2. Capitalization condition: Asset reaches predetermined usable state
3. Capitalization amount includes all related costs (purchase price, transportation, installation, commissioning, etc.)
4. Automatically generate accounting vouchers (Debit: Fixed Asset, Credit: CIP)
5. Support partial capitalization (phased usage)
6. Capitalization date determines depreciation start date
7. Capitalization approval workflow
8. Capitalization variance analysis report

## Preconditions
- Asset receiving completed
- CIP record exists
- User has "Capitalization" permission
- Accounting period is open

## Postconditions
- Asset status changes to "In Use"
- Depreciation calculation starts
- Accounting voucher generated
- CIP amount cleared

## Input Data
1. Asset ID or CIP project number
2. Capitalization date
3. Capitalization amount
4. Capitalization reason
5. Approver
6. Accounting period
7. Cost allocation details

## Output Data
1. Capitalization record
2. Accounting voucher number
3. Asset status update
4. Depreciation plan generated
5. Capitalization completion report

## Validation Rules
1. Capitalization date must be within current accounting period
2. Capitalization amount cannot exceed CIP accumulated cost
3. Asset must be in "Received Complete" or "In Construction" status
4. Accounting period must be open
5. Approval workflow must be completed

## Error Handling
1. CIP cost insufficient: Show insufficient cost and suggest adjustment
2. Accounting period closed: Return error "Accounting period closed"
3. Approval not completed: Block capitalization operation
4. Depreciation rule not configured: Prompt to configure depreciation policy first

## Priority
High
