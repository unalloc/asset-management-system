**Regulatory Compliance Details:**

**WEEE Directive (EU) 2012/19/EU:**
- Producer responsibility for electronic equipment
- Registration in each EU member state
- Marking with crossed-out wheelie bin symbol
- Annual reporting of collected/recycled amounts
- Recycling certificates required

**EPA/RCRA (US) 40 CFR Parts 260-270:**
- Hazardous waste tracking from cradle to grave
- EPA Form 8700-22 manifest required
- EPA ID for handlers
- Universal waste rules for batteries, lamps, electronics

**RoHS Directive (EU) 2011/65/EU:**
- Restricted substances: Lead, Mercury, Cadmium, Cr6+, PBB, PBDE
- Maximum concentration: 0.1% (0.01% for Cadmium)
- Article-level compliance documentation

**REACH (EC 1907/2006):**
- SVHC (Substances of Very High Concern) tracking
- Article-level notification for >0.1% SVHC
- Supplier communication requirements

## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Asset contains hazardous substances | Check Asset.HazardousMaterial = TRUE OR Asset.Category IN ('ELECTRONICS', 'BATTERIES') | EDC-101 | "Asset requires environmental review" |
