# F-2.6.18 - Asset Residual Value Realization

**Requirement ID:** F-2.6.18  
**Module:** Asset Lifecycle Management  
**Priority:** Medium

---

## Requirement Description

Track actual residual value realization after asset disposal, including sales revenue, recovery revenue, cost savings, compared with expected residual value analysis.

---

## Business Rules

1. Asset disposal must be completed before residual value tracking
2. Disposal proceeds must be recorded
3. Expected residual value must exist for comparison
4. Realization Rate = (Actual Residual Value / Expected Residual Value) × 100
5. Variance = Actual Value - Expected Value
6. Gain/Loss on Disposal = Proceeds - Carrying Value (Acquisition Cost - Accumulated Depreciation)
7. Low realization (< expected by threshold) triggers warning

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| AssetID | Yes | UUID of disposed asset |
| ActualValue | Yes | Actual residual value realized |
| SalePrice | Yes | Disposal sale price |
| CostSavings | No | Avoided disposal costs |
| BuyerInfo | No | Buyer information |
| RealizationDate | Yes | Date value was realized |

---

## Output Data

| Data | Description |
|------|-------------|
| ResidualValueRecordID | System-generated UUID |
| VarianceAmount | Difference between actual and expected |
| RealizationRate | Percentage of expected value achieved |
| GainLossOnDisposal | Financial result of disposal |

---

## Validation Rules

1. Asset disposal must be completed before recording
2. Disposal proceeds must be recorded
3. Expected residual value must be set
4. Actual value must be positive

---

## Access Control

| Role | Permission |
|------|------------|
| SystemAdmin | Full access to residual value tracking |
| FinanceManager | View and analyze residual value data |
| AssetManager | Record disposal proceeds |

---

## Multi-Company Data Scope

- **Data Level**: Company/Division
- **Access**: Residual value data at company/division level
- **Isolation**: CompanyCode filtering on value records

---

## Compliance Requirements

- US GAAP: ASC 360 - gain/loss recognition (proceeds - carrying value); residual value estimation reviewed annually
- IFRS: IAS 16 - same requirements as US GAAP
- IFRS 16: Residual value for ROU assets tracked
- SOX: Disposal gain/loss tracking with audit trail
