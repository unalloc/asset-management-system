## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Lease contract approaching expiration | Check Lease.ExpirationDate within 90 days | LSE-101 | "Lease not approaching expiration" |
| PRE-002 | Purchase/return decision made | Check LeaseDecision exists | LSE-102 | "Lease decision not made" |
| PRE-003 | Financial resources available | Check Budget.Approved for purchase | LSE-103 | "Purchase budget not approved" |

### 3.3 Business Logic

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Purchase Price | Decision = 'PURCHASE' | Calculate FMV or contract price | 1 |
| BR-002 | Return Inspection | Decision = 'RETURN' | Create inspection record | 1 |
| BR-003 | ROU Asset Removal | Lease terminated | Remove ROU asset from books | 1 |

**Calculation Logic:**
```
IF Decision = 'PURCHASE' THEN
  PurchasePrice = LeaseContract.PurchaseOptionPrice
  OR IF FMV_Required THEN
    PurchasePrice = FairMarketValue (independent appraisal)
```
