# Module 2.6: Asset Lifecycle Management

## Overview
This module handles the complete lifecycle of assets from procurement to disposal, including procurement workflow integration, asset receiving and inspection, capitalization process, deployment and commissioning, borrowing and return, inventory counting, asset disposal, warranty claims, and lease management.
