### 1.1 Requirement Description

Manage inter-company asset leasing within the group, including lease contract management, rent calculation, expense settlement, tax treatment, and accounting treatment.

**Regulatory Compliance Details:**

**IFRS 16/ASC 842:**
- All leases (except short-term < 12 months, low-value) create ROU asset and lease liability
- Lease liability at PV of remaining payments
- Interest expense + depreciation on ROU asset
- Disclosure requirements for lease term, payments, etc.

## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|

### 3.2 Post-conditions

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Lease contract effective | LeaseContract record created |
| POST-002 | ROU asset created (if applicable) | Asset record created for lessee |
| POST-003 | Lease liability recorded | Liability entry created |
| POST-004 | Payment schedule active | Payment schedule generated |

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Validate lessor owns asset and can lease
  │
  ▼
Determine lease classification
  │
  ├─► FINANCE LEASE (ownership transfers OR substantially all risks)
  │     ├─► Calculate PV of lease payments
  │     ├─► Create ROU asset at lower of PV or fair value
  │     └─► Create lease liability
  │
  └─► OPERATING LEASE (all other leases)
        ├─► Record straight-line expense
        └─► No ROU asset or liability recognition
  │
  ▼
Is short-term lease (< 12 months)?
  └─► YES ─► Apply short-term lease exception
  │
  ▼
Is low-value lease (< $5,000)?
  └─► YES ─► Apply low-value exception
  │
  ▼
Calculate arm's length rent (if inter-company)
  │
  ▼
Generate payment schedule
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Finance Lease Classification | LeaseType = 'FINANCE' OR transfer-of-ownership | Create ROU asset + liability | 1 |
| BR-002 | Short-term Lease Exception | Term < 12 months | Exclude from balance sheet | 1 |
| BR-003 | Low-value Lease Exception | AssetValue < 5,000 USD | Expense immediately | 1 |
| BR-004 | Arm's Length Rent | Inter-company lease | Require market rent documentation | 1 |
| BR-005 | Lease Incentive Treatment | Rent-free period, tenant improvement | Adjust lease payments | 2 |
| BR-006 | Variable Rent | Variable payments based on usage | Exclude from liability calculation | 1 |

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-001 | Lease modification mid-term | Remeasure lease liability | Handle as new lease or modification |
| EC-002 | Asset returned early | Calculate termination gain/loss | Reconcile payments |
| EC-003 | Renewal option exercise | Update lease term if reasonably certain | Remeasure if exercise |

### 3.5 State Transitions

```
Lease Contract: DRAFT → ACTIVE → EXPIRED → TERMINATED
                      │
                      ▼
                 UNDER_RENEGOTIATION
```
