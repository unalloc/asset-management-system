# F-2.8.4 - Group Consolidated Reporting

## Requirement ID
F-2.8.4

## Module
Multi-Company Module

## Priority
High

## Requirement Description
Generate group consolidated reports including asset summary, financial status, and operational performance, supporting multi-currency conversion, inter-company transaction elimination, and minority interest handling.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Full Consolidation | Ownership >= 50% OR control | Combine 100% assets/liabilities | 1 |
| BR-002 | Equity Method | Ownership 20-50% | Record investment at cost +/- earnings | 1 |
| BR-003 | Cost Method | Ownership < 20% | Record at cost | 2 |
| BR-004 | NCI Calculation | Non-wholly-owned subsidiary | Calculate NCI share | 1 |
| BR-005 | Intercompany Elimination | Matching I/C transaction | Generate elimination entry | 1 |
| BR-006 | Currency Translation | Foreign entity | Apply temporal/current rate method | 1 |

**Consolidation Methods:**
- Full Consolidation: 100% combine + NCI (>= 50% or control)
- Proportionate Consolidation: Ownership % combine
- Equity Method: Investment at cost +/- earnings (20-50%)
- Cost Method: Cost basis (< 20%)

**Pre-conditions:**

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Company financial data available | All entities in scope have submitted data | CR-001 | "Missing financial data from {Company}" |
| PRE-002 | Exchange rate data available | ExchangeRates for period and currency pair exist | CR-002 | "Missing exchange rate for {CurrencyPair}" |
| PRE-003 | Ownership ratio data accurate | Ownership percentages sum correctly | CR-003 | "Ownership percentages do not sum to 100%" |
| PRE-004 | User has consolidation access | User.Role IN ('GROUP_CONTROLLER', 'SYSTEM_ADMIN') | CR-004 | "Insufficient permissions for consolidation" |

**Post-conditions:**

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Group consolidated report generated | Report with all entities |
| POST-002 | Consolidation adjustments recorded | Adjustment entries created |
| POST-003 | Audit tracking saved | Full audit trail |

**Edge Cases:**

| Edge Case ID | Scenario | Expected Behavior |
|--------------|----------|-------------------|
| EC-001 | Missing exchange rate | Use most recent available + flag warning |
| EC-002 | Unmatched intercompany | Flag for manual investigation |
| EC-003 | Entity in loss position | Special NCI calculation may apply |
| EC-004 | Hyperinflationary economy | IAS 29 restatement treatment |
