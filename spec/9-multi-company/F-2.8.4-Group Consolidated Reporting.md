### 1.1 Requirement Description

Generate group consolidated reports including asset summary, financial status, and operational performance, supporting multi-currency conversion, inter-company transaction elimination, and minority interest handling.

## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Company financial data available | All entities in scope have submitted data | CR-001 | "Missing financial data from {Company}" |

| Post-condition ID | Description | Success Criteria | Verification Method |
|-------------------|-------------|-------------------|---------------------|

| POST-002 | Consolidation adjustments recorded | Adjustment entries created | Audit log |
| POST-003 | Audit tracking saved | Full audit trail | Audit query |

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Select consolidation scope (all entities or specific)
  │
  ▼
Fetch entity financial data for period
  │
  ▼
Fetch applicable exchange rates
  │
  ▼
For each inter-company transaction:
  │   Match debit/credit between entities
  │   Flag unmatched items for investigation
  │
  ▼
Calculate non-controlling interest (NCI)
  │
  ▼
Apply currency translation
  │
  ▼
Generate elimination entries
  │
  ▼
Combine entity financials (by consolidation method)
  │
  ▼
Generate consolidated statements
  │
  ▼
Create audit trail
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Full Consolidation | Ownership >= 50% OR control | Combine 100% assets/liabilities | 1 |
| BR-002 | Equity Method | Ownership 20-50% | Record investment at cost +/- earnings | 1 |
| BR-003 | Cost Method | Ownership < 20% | Record at cost | 2 |
| BR-004 | NCI Calculation | Non-wholly-owned subsidiary | Calculate NCI share | 1 |
| BR-005 | Intercompany Elimination | Matching I/C transaction | Generate elimination entry | 1 |
| BR-006 | Currency Translation | Foreign entity | Apply temporal/current rate method | 1 |

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-001 | Missing exchange rate | Use most recent available + flag | CR-005 warning |
| EC-002 | Unmatched intercompany | Investigate manually | Flag for resolution |
| EC-003 | Entity in loss position | NCI may exceed ownership | Special calculation |
| EC-004 | Hyperinflationary economy | Restate in stable currency | IAS 29 treatment |

### 3.5 State Transitions

```
States: DRAFT → VALIDATED → APPROVED → PUBLISHED → ARCHIVED

DRAFT ──► VALIDATED ──► APPROVED ──► PUBLISHED ──► ARCHIVED
  │          │             │            │             │
  ▼          ▼             ▼            ▼             ▼
 Data      Balance      Controller   Distribution   Historical
 Entry     Check         Sign-off     Complete       Record
```
