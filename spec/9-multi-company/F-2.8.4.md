# F-2.8.4 Group Consolidated Reporting

## Function ID
F-2.8.4

## Function Name
Group Consolidated Reporting

## Function Description
Generate group consolidated reports including asset summary, financial status, and operational performance, supporting multi-currency conversion, inter-company transaction elimination, and minority interest handling.

## Business Rules
1. Report scope: Generate reports by organizational level (group, holding company, business segment)
2. Currency conversion: Multi-currency data converted to report currency (group base currency) using appropriate exchange rates
3. Inter-company elimination: Eliminate inter-company transactions (asset transfer, leasing, services) to avoid double counting
4. Minority interest: Handle minority interest share of non-wholly-owned subsidiaries
5. Report period: Monthly, quarterly, annual consolidated reports
6. Report versions: Actual, budget, forecast, year-over-year, quarter-over-quarter
7. Audit tracking: Complete audit of consolidation adjustment entries, exchange rate usage, elimination calculations
8. Disclosure requirements: Meet financial report disclosure requirements of different countries/regions

## Preconditions
- Each company's financial and asset data available
- Exchange rate data available
- Inter-company transaction data complete
- Ownership ratio data accurate
- Report period defined

## Postconditions
- Group consolidated report generated
- Consolidation adjustment entries recorded
- Audit tracking saved
- Report distribution and archiving

## Input Data
1. Each company's financial and asset data
2. Exchange rate data (historical rate, average rate, period-end rate)
3. Inter-company transaction details
4. Ownership ratio data
5. Report period and scope
6. Report currency
7. Consolidation rules (elimination rules, minority interest treatment methods)
8. Disclosure requirement templates

## Output Data
1. Group consolidated report (assets, liabilities, equity, revenue, expenses, etc.)
2. Consolidation work papers
3. Consolidation adjustment entries
4. Currency conversion calculation table
5. Inter-company transaction elimination table
6. Minority interest calculation table
7. Audit tracking report
8. Disclosure notes

## Validation Rules
1. Data must be complete (all companies within consolidation scope)
2. Exchange rates must be applicable to corresponding period
3. Inter-company transactions must match (amount, period)
4. Ownership ratios must be accurate
5. Consolidation rules must comply with accounting standards

## Error Handling
1. Data missing: Mark report as incomplete, show data gaps
2. Exchange rate missing: Use most recent available rate and mark
3. Transaction mismatch: Investigate differences and manually adjust
4. Rule conflict: Accounting department determines applicable rule

High
