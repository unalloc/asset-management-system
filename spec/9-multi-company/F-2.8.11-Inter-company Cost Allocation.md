### 1.1 Requirement Description

Manage inter-company cost allocation within the group, including shared service costs, centralized procurement costs, and group management fees, supporting fair and reasonable cost sharing and settlement.

## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Cost data available | CostPool.Amount > 0 | ICA-001 | "No costs to allocate" |
| PRE-002 | Allocation basis defined | AllocationBasis IN ('HEADCOUNT', 'REVENUE', 'ASSETS', 'USAGE') | ICA-002 | "Allocation basis not defined" |
| PRE-003 | Beneficiary companies identified | CompanyList NOT EMPTY | ICA-003 | "No beneficiary companies" |
| PRE-004 | User has allocation rights | User.Role IN ('GROUP_FINANCE', 'SYSTEM_ADMIN') | ICA-004 | "Insufficient allocation rights" |

### 3.2 Post-conditions

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Cost allocation calculated | Allocation entries created |
| POST-002 | Inter-company invoices generated | Invoices for each beneficiary |
| POST-003 | Audit trail created | Full documentation of allocation |

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Identify costs to allocate
  │
  ▼
Determine allocation basis
  │
  ├─► HEADCOUNT ─► Allocate by employee count
  ├─► REVENUE ─► Allocate by revenue proportion
  ├─► ASSETS ─► Allocate by asset value
  └─► USAGE ─► Allocate by measured usage
       │
       ▼
  Calculate allocation ratios
       │
       ▼
  Apply ratios to total cost
       │
       ▼
  Generate inter-company invoices
       │
       ▼
  Record in beneficiary companies
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Benefit Test | Any allocation | Require documented benefit to recipient | 1 |
| BR-002 | Arm's Length Allocation | Inter-company | Require reasonable basis documentation | 1 |
| BR-003 | Allocation Documentation | Any allocation > threshold | Create TP documentation | 1 |
| BR-004 | Budget Integration | Annual allocation | Link to approved budget | 2 |
| BR-005 | Dispute Resolution | Recipient challenges | Require justification + appeal process | 2 |

### 3.4 Allocation Methods

| Method | Description | Best For |
|--------|-------------|----------|
| Direct Allocation | Allocate directly to beneficiaries | Direct services |
| Step Allocation | Sequential allocation of shared costs | Cascading services |
| Reciprocal Allocation | Simultaneous solution for mutual services | Intercompany services |
