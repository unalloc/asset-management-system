## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Group configuration defined | GroupConfig NOT NULL | CC-001 | "Group configuration not defined" |
| PRE-002 | Company exists | CompanyCode VALID | CC-002 | "Company not found" |
| PRE-003 | User has configuration rights | User.Role IN ('GROUP_CONFIG_ADMIN', 'COMPANY_ADMIN', 'SYSTEM_ADMIN') | CC-003 | "Insufficient configuration rights" |

### 3.2 Post-conditions

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Company config effective | Config values override group defaults |
| POST-002 | Inheritance chain validated | No circular inheritance |
| POST-003 | Change audit logged | Configuration change tracked |

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Is user authorized?
    ├─► NO ─► Return CC-003
    └─► YES
         │
         ▼
  Select configuration scope
         │
         ▼
  Apply inheritance (Group → Region → Country → Company)
         │
         ▼
  Validate company override
         │
         ▼
  Is override within allowed range?
    ├─► NO ─► Return CC-004
    └─► YES
         │
         ▼
  Save configuration
         │
         ▼
  Log change to audit trail
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Inheritance Priority | Any configuration | Group > Region > Country > Company | 1 |
| BR-002 | Allowed Override Range | Company modification | Must be within min/max from parent | 1 |
| BR-003 | Configuration Version | Any change | Create new version | 1 |
| BR-004 | No Core Override | Group core principles | Cannot be overridden at company level | 1 |

### 3.4 Configuration Types
