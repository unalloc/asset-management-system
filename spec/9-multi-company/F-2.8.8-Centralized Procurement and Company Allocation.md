# F-2.8.8 - Centralized Procurement and Company Allocation

## Requirement ID
F-2.8.8

## Module
Multi-Company Module

## Priority
High

## Requirement Description
Support group centralized procurement of assets, then allocation to subsidiaries, including procurement negotiation, contract signing, asset receiving, internal allocation, and cost sharing.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Demand Aggregation | Multiple company requests | Combine demand for volume discount | 1 |
| BR-002 | Supplier Approval | Any procurement | REQUIRE Supplier.ApprovedFlag | 1 |
| BR-003 | Cost Allocation Arm's Length | Inter-company allocation | Require allocation basis documentation | 1 |
| BR-004 | Asset Tracking | Centralized asset receipt | Track from PO to allocation | 1 |
| BR-005 | Inter-company Billing | Asset allocated | Generate I/C invoice to subsidiary | 2 |

**Allocation Methods:**
- FIXED_RATIO: Fixed percentage allocation
- USAGE_BASED: Based on usage metrics
- BENCHMARK: Based on benchmarking

**Pre-conditions:**

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Procurement demand aggregated | DemandList NOT EMPTY | CP-001 | "No procurement demand submitted" |
| PRE-002 | Supplier selected | Supplier.Status = 'APPROVED' | CP-002 | "Supplier not approved" |
| PRE-003 | Allocation rules defined | AllocationMethod IN ('FIXED_RATIO', 'USAGE_BASED', 'BENCHMARK') | CP-003 | "Allocation method not defined" |
| PRE-004 | User has procurement rights | User.Role IN ('GROUP_PROCUREMENT', 'SYSTEM_ADMIN') | CP-004 | "Insufficient procurement rights" |

**Post-conditions:**

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Procurement contract signed | Contract record with terms |
| POST-002 | Assets allocated to companies | Allocation records created |
| POST-003 | Cost sharing calculated | Cost allocation entries posted |

**Edge Cases:**

| Edge Case ID | Scenario | Expected Behavior |
|--------------|----------|-------------------|
| EC-001 | Demand exceeds contract quantity | Negotiate additional or allocate proportionally |
| EC-002 | Supplier delivery delayed | Notify subsidiaries, adjust timelines |
| EC-003 | Company refuses allocated asset | Reallocate or cancel, dispute resolution |
