## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Transferring company owns asset | Asset.CompanyCode = TransferFromCompanyCode | CT-001 | "Asset not owned by transferring company" |
| PRE-002 | Asset can be transferred | Asset.Status NOT IN ('MORTGAGED', 'LIENS', 'RESTRICTED') | CT-002 | "Asset has transfer restrictions" |
| PRE-003 | Receiving company can receive | ReceivingCompany.Status = 'ACTIVE' | CT-003 | "Receiving company not active" |
| PRE-004 | Transfer pricing defined | TransferPrice IS NOT NULL OR PriceMethod = 'MARKET' | CT-004 | "Transfer price or method required" |
| PRE-005 | Approval workflow complete | All required approvals obtained | CT-005 | "Transfer approval incomplete" |

### 3.2 Post-conditions

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Asset ownership changed | Asset.CompanyCode = TransferToCompanyCode |
| POST-002 | Financial entries posted | GL entries created |
| POST-003 | Tax filings updated | Tax records updated |
| POST-004 | Asset history updated | Transfer record in asset history |

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Is asset transferable?
  ├─► NO ─► Return CT-002 with restriction reason
  └─► YES
       │
       ▼
  Determine transfer type
       │
       ├─► PAID (Internal Sale)
       │     └─► Determine arm's length price
       │
       ├─► UNPAID (Donation)
       │     └─► Calculate deemed cost
       │
       ├─► LEASE TRANSFER
       │     └─► Update lease terms
       │
       └─► TEMPORARY BORROWING
             └─► Set return conditions
       │
       ▼
  Execute approval workflow
       │
       ▼
  Is approved?
    ├─► NO ─► Return CT-006 with rejection reason
    └─► YES
         │
         ▼
    Post financial entries
         │
     ▼
  Update tax records
         │
         ▼
  Transfer asset ownership
         │
         ▼
  Create audit trail
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Arm's Length Price | TransferType = 'PAID' | REQUIRE price >= min_arm_length AND <= max_arm_length | 1 |
| BR-002 | Depreciation Recalculation | Any transfer | Calculate NBV at transfer date | 1 |
| BR-003 | Tax Basis Transfer | EU cross-border | Reset tax basis per jurisdiction | 2 |
| BR-004 | Gain/Loss Recognition | Any paid transfer | Calculate and recognize gain/loss | 1 |
| BR-005 | Asset History Tracking | Any transfer | Create transfer record with full history | 1 |
| BR-006 | Intercompany Receivable | Paid transfer | Create I/C receivable/payable | 2 |

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-001 | Partially depreciated asset | Transfer NBV = Cost - Accumulated Depreciation | Calculate correctly |
| EC-002 | Asset under lease transfer | Lease assignment required | Validate lease consent |
| EC-003 | Cross-border with different currencies | Convert at spot rate | Apply exchange rate |
| EC-004 | Asset with pending maintenance | Complete or transfer obligations | Decide before transfer |

### 3.5 State Transitions

```
Asset States: ACTIVE ──► TRANSFER_PENDING ──► TRANSFERRED ──► ACTIVE
                                  │
                                  ▼
                           TRANSFER_REJECTED

Transfer Request: DRAFT → PENDING_APPROVAL → APPROVED → EXECUTED
                         │
                         ▼
                     REJECTED
```
