# F-2.8.13 - Support Different Company Types

## Requirement ID
F-2.8.13

## Module
Multi-Company Module

## Priority
High (recommended upgrade from Medium)

## Requirement Description
Support different types of company entities, including wholly-owned subsidiaries, controlling subsidiaries, minority-owned companies, joint ventures, and branch offices, with different management requirements and integration levels for each type.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Wholly-Owned Consolidation | CompanyType = 'WHOLLY_OWNED' | Full consolidation (100%) | 1 |
| BR-002 | Controlling Entity Consolidation | CompanyType = 'CONTROLLING' AND Ownership > 50% | Full consolidation + NCI | 1 |
| BR-003 | Minority Interest Calculation | Non-wholly-owned subsidiary | Calculate NCI share | 1 |
| BR-004 | Equity Method Application | Ownership 20-50% OR significant influence | Equity method | 1 |
| BR-005 | Cost Method Application | Ownership < 20% AND no influence | Cost method | 2 |
| BR-006 | Joint Venture Treatment | CompanyType = 'JOINT_VENTURE' | Proportionate or equity | 1 |

**Company Types:**

| Type | Ownership | Control | Consolidation | Data Access |
|------|-----------|---------|---------------|--------------|
| Wholly-Owned | 100% | Full | Full | Full integration |
| Controlling | >50% | Full | Full + NCI | Full integration |
| Minority | ≤50% | Limited | Equity/Cost | Limited integration |
| Joint Venture | Variable | Shared | Proportionate/Equity | Interface integration |
| Branch | N/A (non-entity) | Full | Integrated | Full integration |

**Pre-conditions:**

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Company type determined | CompanyType IS NOT NULL | CT-001 | "Company type must be specified" |
| PRE-002 | Ownership data available | OwnershipPercentage NOT NULL | CT-002 | "Ownership percentage required" |
| PRE-003 | Consolidation method determined | ConsolidationMethod NOT NULL | CT-003 | "Consolidation method required" |

**Post-conditions:**

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Company type configured | Type-specific settings applied |
| POST-002 | Consolidation method set | Appropriate method per ownership |
| POST-003 | Integration level established | Data flows configured per type |

## Multi-Company Data Scope
Group configures all company type settings. Branches inherit from parent company. Consolidation methods determined by group based on ownership and control structure.
