### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Define benchmarking scope and peer group
  │
  ▼
Collect performance data from companies
  │
  ▼
Standardize data (adjust for size, currency, etc.)
  │
  ▼
Calculate benchmarking metrics
  │
  ├─► Utilization Rate
  ├─► Maintenance Cost Ratio
  ├─► MTBF (Mean Time Between Failures)
  ├─► Lifecycle Cost per Unit
  └─► Asset Availability
       │
       ▼
  Rank companies by metric
       │
       ▼
  Identify best performers
       │
       ▼
  Analyze practices of top performers
       │
       ▼
  Generate improvement suggestions
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Data Anonymization | Sharing sensitive data | Anonymize company identifiers | 1 |
| BR-002 | Peer Group Validity | Benchmarking group | Require minimum 3 comparable companies | 2 |
| BR-003 | Metric Consistency | Any metric calculation | Require standardized definitions | 1 |
| BR-004 | Statistical Significance | Small sample size | Flag when n < 5 | 2 |

### 3.4 Benchmarking Metrics

| Metric | Definition | Unit | Data Source |
|--------|------------|------|-------------|
| Asset Utilization | Actual usage / Maximum capacity | % | Asset tracking |
| Maintenance Cost Ratio | Maintenance cost / Asset value | % | Work orders |
| MTBF | Total operating time / Failure count | Hours | Maintenance records |
| Lifecycle Cost | Total cost over life / Life years | $/year | Financial data |
| Availability | Uptime / Total time | % | Asset tracking |
