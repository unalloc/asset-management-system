# F-2.8.15 - Group Asset Optimization Suggestions

## Requirement ID
F-2.8.15

## Module
Multi-Company Module

## Priority
High

## Requirement Description
Propose asset optimization suggestions based on group overall perspective, including asset redistribution, sharing, disposal, and procurement, to maximize group overall asset value.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Group-Optimum Not Company-Optimum | Any optimization | Maximize group value, not single company | 1 |
| BR-002 | Feasibility Assessment | Any suggestion | Include cost/benefit/risk analysis | 1 |
| BR-003 | Stakeholder Impact | Cross-entity suggestion | Require affected company input | 2 |
| BR-004 | Implementation Tracking | Suggestion accepted | Monitor actual vs projected benefits | 1 |

**Optimization Types:**

| Type | Description | Key Metrics |
|------|-------------|-------------|
| Redistribution | Move assets to higher-utilization entities | Utilization rate, relocation cost |
| Sharing | Enable shared asset pools | Shared usage %, coordination cost |
| Disposal | Identify assets to dispose | NBV, salvage value, disposal cost |
| Procurement | Optimize timing/quantity | Demand forecast, lead time |

**Pre-conditions:**

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Group asset panoramic data available | All entity asset data accessible | OS-001 | "Asset data not accessible" |
| PRE-002 | Optimization model configured | ModelType IN ('REDISTRIBUTION', 'SHARING', 'DISPOSAL', 'PROCUREMENT') | OS-002 | "Optimization model not configured" |
| PRE-003 | User has optimization rights | User.Role IN ('GROUP_ASSET_MANAGER', 'SYSTEM_ADMIN') | OS-003 | "Insufficient optimization rights" |

**Post-conditions:**

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Optimization analysis complete | Analysis report generated |
| POST-002 | Suggestions generated | Actionable recommendations created |
| POST-003 | Tracking mechanism established | Implementation monitoring active |

## Multi-Company Data Scope
Group Asset Manager creates and views all optimization analyses. Company Managers can only view the impact on their own company. Optimization focuses on maximizing group value, not individual company value.
