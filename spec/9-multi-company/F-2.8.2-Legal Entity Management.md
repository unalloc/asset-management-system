# F-2.8.2 - Legal Entity Management

## Requirement ID
F-2.8.2

## Module
Multi-Company Module

## Priority
High

## Requirement Description
Manage legal entities (companies) within the group, including company registration information, legal status, tax information, and independent book settings.

## Business Rules

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-001 | Ownership Sum Validation | Sum of all subsidiary ownership <= 100% | THROW ERROR if sum > 100% | 1 |
| BR-002 | Consolidation Method Assignment | Based on ownership percentage | AUTO-ASSIGN: 100%=Full, 50-99%=Full, 20-50%=Equity, <20%=Cost | 1 |
| BR-003 | Joint Venture Control | CompanyType = 'JOINT_VENTURE' | REQUIRE JointControlFlag = TRUE AND exactly 2-5 owners | 1 |
| BR-004 | Tax Registration Validation | Jurisdiction IN EU | REQUIRE VAT registration for EU entities | 2 |
| BR-005 | Registration Number Format | Company created in specific country | VALIDATE against country-specific format | 2 |
| BR-006 | Business Term Validation | BusinessTermEndDate provided | REQUIRE BusinessTermEndDate > BusinessTermStartDate | 1 |

**Entity Types:**
- WHOLLY_OWNED: 100% ownership
- CONTROLLING: >50% ownership
- MINORITY: <50% ownership
- JOINT_VENTURE: Joint control with 2-5 owners

**Consolidation Methods:**
- Full Consolidation (100% or 50-99% with control)
- Proportionate Consolidation (Joint ventures)
- Equity Method (20-50% ownership)
- Cost Method (<20% ownership)

**Pre-conditions:**

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-001 | Legal entity legal information available | CompanyName, RegistrationNumber, JurisdictionCode NOT NULL | LE-001 | "Legal entity basic information required" |
| PRE-002 | Accounting standards defined | AccountingStandardCode IN ('US_GAAP', 'IFRS', 'LOCAL_GAAP') | LE-002 | "Valid accounting standard must be selected" |
| PRE-003 | User has legal entity management permission | User.RolePermissions CONTAINS 'LEGAL_ENTITY_ADMIN' | LE-003 | "User lacks legal entity management permission" |
| PRE-004 | Tax rules database available | TaxRules.TaxRegion EXISTS for JurisdictionCode | LE-004 | "Tax rules not available for jurisdiction" |
| PRE-005 | Group organization exists | EXISTS(Organization WHERE OrganizationLevel = 'GROUP') | LE-005 | "Group organization must exist before creating legal entity" |

**Post-conditions:**

| Post-condition ID | Description | Success Criteria |
|-------------------|-------------|-------------------|
| POST-001 | Legal entity record created | Company record with unique CompanyCode |
| POST-002 | Independent book settings configured | CompanyCode linked to accounting configuration |
| POST-003 | Tax configuration completed | Tax rates configured for applicable taxes |
| POST-004 | Ownership relationship recorded | Ownership record created with valid percentages |

**Edge Cases:**

| Edge Case ID | Scenario | Expected Behavior |
|--------------|----------|-------------------|
| EC-001 | Ownership changes from 49% to 51% | Trigger consolidation method change, generate audit entry |
| EC-002 | Minority becomes controlling | Trigger full consolidation setup, require new control documentation |
| EC-003 | Entity dissolution/liquidation | Set status = 'LIQUIDATION', prevent new transactions |
| EC-004 | Joint venture converts to subsidiary | Update company type, recalculate ownership |
| EC-005 | Multiple registrations in same country | Allow multiple legal entities per country |

## Multi-Company Data Scope
Only Group-level roles can create/modify legal entities. Company-level users can view their own entity only. CompanyCode scope isolation between legal entities.
