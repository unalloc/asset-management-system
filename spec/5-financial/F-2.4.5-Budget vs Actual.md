## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-BUD-001 | Budget must be positive | BudgetAmount > 0 | BUD-ERR-001 | "Budget amount must be positive" |
| PRE-BUD-002 | Budget period valid | StartDate < EndDate | BUD-ERR-002 | "Budget period end must be after start date" |
| PRE-BUD-003 | Cost center must exist | CostCenterID IN (SELECT CostCenterID FROM CostCenter) | BUD-ERR-003 | "Invalid cost center" |
| PRE-BUD-004 | Budget account must be valid | AccountID IN (SELECT AccountID FROM ChartOfAccounts) | BUD-ERR-004 | "Invalid budget account" |
| PRE-BUD-005 | Approval workflow complete | ApprovalStatus = 'APPROVED' | BUD-ERR-005 | "Budget must be approved before use" |

**Pre-condition Validation Matrix:**

```
IF (Budget.BudgetAmount > 0)
   AND (Budget.StartDate < Budget.EndDate)
   AND (Budget.CostCenterID IS NOT NULL)
   AND (Budget.AccountID IS NOT NULL)
   AND (Budget.ApprovalStatus = 'APPROVED')
THEN activate budget for tracking
ELSE throw BUD-ERR-00X Error
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-BUD-001 | Variance Calculation | Standard | Variance = BudgetAmount - ActualAmount | 1 |
| BR-BUD-002 | Variance Percentage | Standard | Variance% = (Variance / BudgetAmount) × 100 | 1 |
| BR-BUD-003 | Alert Threshold | Variance% > Threshold | Generate alert | 1 |
| BR-BUD-004 | Budget Check | Overrun detected | Block or warn based on policy | 1 |
| BR-BUD-005 | Carryforward Calculation | Policy = CARRYFORWARD | RemainingBudget × CarryforwardRate | 1 |
| BR-BUD-006 | Commitment Accounting | POs issued not yet received | Track as committed | 2 |
| BR-BUD-007 | Multi-period Budget | Period spread | Allocate across periods | 2 |

**Calculation Logic:**

```
Budget Available = BudgetAmount - ActualAmount - CommittedAmount

Variance = BudgetAmount - ActualAmount

Variance Percentage = (Variance / BudgetAmount) × 100

Alert Condition:
  IF (ABS(Variance%) > VarianceThreshold) THEN SendAlert

Carryforward (if approved):
  CarryforwardAmount = MAX(0, BudgetAmount - ActualAmount) × CarryforwardRate
```
