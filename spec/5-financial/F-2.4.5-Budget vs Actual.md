# F-2.4.5 - Budget vs Actual

**Requirement ID:** F-2.4.5  
**Module:** Financial Management  
**Priority:** Medium

---

## Requirement Description

Compare asset-related budgets with actual expenditures, perform variance analysis and control. Track maintenance budgets, capital budgets, and operating budgets against actual spending with alerts for over-budget conditions.

---

## Business Rules

1. Variance calculated as Budget Amount minus Actual Amount
2. Variance percentage = (Variance / Budget Amount) x 100
3. Alerts generated when variance exceeds defined threshold
4. Budget overrun requires approval before exceeding budget
5. Carryforward of unused budget allowed per policy
6. Commitment accounting for POs issued but not yet received

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Budget Code | Yes | Unique identifier for the budget |
| Budget Amount | Yes | Planned expenditure |
| Cost Center | Yes | Department or cost center |
| Account | Yes | Chart of accounts code |
| Start Date | Yes | Budget period beginning |
| End Date | Yes | Budget period end |
| Variance Threshold | Yes | Percentage that triggers alert |

---

## Output Data

| Data | Description |
|------|-------------|
| Budget Record | Approved budget with parameters |
| Actuals | Actual expenditures posted |
| Variance Report | Budget vs actual comparison |
| Alerts | Notifications when thresholds exceeded |

---

## Validation Rules

1. Budget amount must be positive
2. Start date must be before end date
3. Cost center must exist
4. Account must be valid
5. Budget must be approved before tracking begins
6. Over-budget requires approval

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full access |
| Company Controller | Create, approve, modify budgets |
| Division Manager | View and transfer within division |
| Plant Manager | View within plant |

---

## Multi-Company Data Scope

- **Data Level**: Company level with division/plant breakdown
- **Access**: Budget creation at company level, viewing progressively restricted
- **Isolation**: Company Code scopes all budget data |

---

## Compliance Requirements

- SOX 404 internal controls over budget vs actual reporting
- Budget approval workflow documentation
- Commitment accounting audit trail
