## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-INVAL-001 | Valuation method valid | Method IN ('REPLACEMENT_COST', 'ACTUAL_CASH_VALUE', 'FUNCTIONAL_REPLACEMENT', ' AGREED_VALUE') | INVAL-ERR-001 | "Invalid valuation method" |
| PRE-INVAL-002 | Cost index available | CostIndex EXISTS FOR AssetCategory AND Period | INVAL-ERR-002 | "Cost index required for valuation" |
| PRE-INVAL-003 | Depreciation parameters valid | DepreciationRate >= 0 AND DepreciationRate <= 1 | INVAL-ERR-003 | "Depreciation rate must be between 0 and 100%" |

**Pre-condition Validation Matrix:**

```
IF (ValuationMethod IN ('REPLACEMENT_COST', 'ACTUAL_CASH_VALUE', 'FUNCTIONAL_REPLACEMENT', 'AGREED_VALUE'))
   AND (CostIndex EXISTS FOR category)
   AND (DepreciationRate >= 0 AND DepreciationRate <= 1)
THEN calculate insurance value
ELSE throw INVAL-ERR-00X Error
```

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Select Valuation Method
  │
  ├─► Replacement Cost ─► Fair value to replace
  ├─► Actual Cash Value ─► Replacement cost minus depreciation
  ├─► Functional Replacement ─► Cost to replace with like function
  └─► Agreed Value ─► Explicitly agreed amount
        │
        ▼
    Apply Adjustments
      │
      ├─► Cost index inflation adjustment
      ├─► Depreciation for ACV
      ├─► Policy limits
      └─► Deductible consideration
            │
            ▼
        Compare to Coverage
          │
          ├─► Insured Value > Coverage ─► Adequate
          ├─► Insured Value < Coverage ─► UNDERINSURED
          └─► Insured Value >> Coverage ─► OVERINSURED
                │
                ▼
            Generate Valuation Report
                │
                ▼
            END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-INVAL-001 | Replacement Cost | Standard | = CurrentCost × CostIndex | 1 |
| BR-INVAL-002 | Actual Cash Value | Standard | = ReplacementCost × (1 - DepreciationRate) | 1 |
| BR-INVAL-003 | Functional Replacement | When applicable | Cost to replace with similar function | 1 |
| BR-INVAL-004 | Agreed Value | Policy specifies | Explicit amount from policy | 1 |
| BR-INVAL-005 | Underinsurance Check | ACV vs Coverage | Flag if Coverage < ACV × 0.8 | 1 |
| BR-INVAL-006 | Overinsurance Check | ACV vs Coverage | Flag if Coverage > ACV × 1.2 | 2 |

**Calculation Logic:**

```
Replacement Cost = OriginalCost × (CurrentIndex / AcquisitionIndex)

Actual Cash Value = ReplacementCost × (1 - Age-Based Depreciation)

Functional Replacement Cost = Estimated cost to replace with equivalent function

Insurance Adequacy Ratio = Coverage Amount / Insured Value
  > 1.0 = Overinsured
  0.8 - 1.0 = Adequate
  < 0.8 = Underinsured

Recommended Coverage = Insured Value × 1.1 (10% contingency)
```

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-INVAL-001 | No cost index | Use manual estimate | Allow manual input |
| EC-INVAL-002 | Obsolete asset | Use functional replacement | Find equivalent |
| EC-INVAL-003 | Partial loss | Apply average depreciation | Pro-rata calculation |
| EC-INVAL-004 | Market fluctuation | Use current market data | Update indexes |
