# F-2.4.14 - Insurance Valuation

**Requirement ID:** F-2.4.14  
**Module:** Financial Management  
**Priority:** Medium

---

## Requirement Description

Apply different insurance valuation methods to determine asset insurance value ensuring adequate coverage. Includes replacement cost, actual cash value, functional replacement, and agreed value approaches with coverage adequacy analysis.

---

## Business Rules

1. Replacement Cost = Original Cost x (Current Index / Acquisition Index)
2. Actual Cash Value = Replacement Cost x (1 - Age-Based Depreciation)
3. Agreed Value = Explicitly agreed amount per policy
4. Adequacy Ratio = Coverage Amount / Insured Value
5. Underinsurance flagged when ratio < 80%
6. Overinsurance flagged when ratio > 120%
7. Recommended coverage = Insured Value x 1.1 (10% contingency)

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Asset | Yes | Asset being valued |
| Valuation Date | Yes | When valuation is performed |
| Valuation Method | Yes | Replacement Cost, ACV, Functional, or Agreed |
| Depreciation Rate | Yes | Age-based depreciation percentage |
| Cost Index | Yes | Inflation adjustment factor |

---

## Output Data

| Data | Description |
|------|-------------|
| Insurance Valuation | Calculated insured value |
| Coverage Gap Analysis | Adequacy assessment |
| Recommended Coverage | Suggested coverage amount |
| Valuation Report | Documentation of valuation |

---

## Validation Rules

1. Valuation method must be valid (Replacement Cost, ACV, Functional, Agreed)
2. Cost index required for valuation
3. Depreciation rate must be between 0 and 100%
4. Insured value cannot be negative
