# F-2.4.2 - Automated Depreciation

**Requirement ID:** F-2.4.2  
**Module:** Financial Management  
**Priority:** High

---

## Requirement Description

Automatically calculate asset depreciation expenses supporting multiple depreciation methods and accounting policies. Includes Straight-Line, Double-Declining Balance, Sum-of-Years Digits, and Units of Production methods with support for US GAAP and IFRS book treatment.

---

## Business Rules

1. Straight-Line depreciation: Annual = (Cost - Salvage) / Useful Life
2. Double-Declining Balance: Rate = (2 / Useful Life), switch to straight-line when beneficial
3. Sum-of-Years Digits: Fraction = (Remaining Life / Sum of Years)
4. Half-year convention applied for mid-year acquisitions in US GAAP
5. Depreciation suspended when asset is idle for more than 30 days
6. Net book value cannot go below residual value
7. Component depreciation for significant parts (IAS 16 / ASC 360 requirement)

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Acquisition Cost | Yes | Original cost of the asset |
| Useful Life | Yes | Expected lifespan in years |
| Residual Value | Yes | Expected value at end of life |
| Depreciation Method | Yes | How depreciation is calculated |
| Depreciation Start Date | Yes | When depreciation begins |
| GAAP Type | Yes | US GAAP or IFRS for book treatment |

---

## Output Data

| Data | Description |
|------|-------------|
| Depreciation Schedule | Period-by-period depreciation amounts |
| Accumulated Depreciation | Total depreciation to date |
| Net Book Value | Current asset value (Cost - Accumulated) |
| Journal Entries | Financial posting records |

---

## Validation Rules

1. Acquisition cost must be positive
2. Useful life must be between 1 and 50 years
3. Residual value must be between 0 and acquisition cost
4. Depreciation start date cannot be more than 30 days in the future
5. Land cannot be depreciated (must use NONE method)
6. Total depreciation cannot exceed cost minus residual value

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full access |
| Company Controller | Create and modify depreciation books |
| Division Finance | View within division |
| Department User | View within department |

---

## Multi-Company Data Scope

- **Data Level**: Company level
- **Access**: Each company maintains its own depreciation books
- **Isolation**: Company Code scopes all depreciation data |

---

## Compliance Requirements

- ASC 360 (US GAAP) depreciation methods
- IAS 16 (IFRS) cost and revaluation models
- IFRS 16 right-of-use asset depreciation
- SOX 404 audit trail for all depreciation calculations
