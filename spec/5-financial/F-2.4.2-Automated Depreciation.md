## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-DEP-001 | Asset cost must be positive | AcquisitionCost > 0 | DEP-ERR-001 | "Acquisition cost must be a positive number" |
| PRE-DEP-002 | Useful life within range | UsefulLife >= 1 AND UsefulLife <= 50 | DEP-ERR-002 | "Useful life must be between 1 and 50 years" |
| PRE-DEP-003 | Residual value rate valid | ResidualValueRate >= 0 AND ResidualValueRate <= 100 | DEP-ERR-003 | "Residual value rate must be between 0 and 100%" |
| PRE-DEP-004 | Depreciation start date valid | DepreciationStartDate <= CURRENT_DATE + 30 days | DEP-ERR-004 | "Depreciation start date cannot be more than 30 days in the future" |
| PRE-DEP-005 | Depreciation method supported | Method IN ('STRAIGHT_LINE', 'DOUBLE_DECLINING', 'SUM_OF_YEARS', 'UNITS_OF_PRODUCTION') | DEP-ERR-005 | "Unsupported depreciation method" |
| PRE-DEP-006 | Asset in depreciable state | Asset.Status IN ('ACTIVE', 'IN_SERVICE') | DEP-ERR-006 | "Asset must be active or in service to depreciate" |

**Pre-condition Validation Matrix:**

```
IF (Asset.AcquisitionCost > 0)
   AND (Asset.UsefulLife >= 1 AND Asset.UsefulLife <= 50)
   AND (Asset.ResidualValueRate >= 0 AND Asset.ResidualValueRate <= 100)
   AND (Asset.Status IN ('ACTIVE', 'IN_SERVICE'))
THEN proceed to depreciation calculation
ELSE throw DEP-ERR-00X Error
```

### 3.2 Post-conditions

| Post-condition ID | Description | Success Criteria | Verification Method |
|-------------------|-------------|-------------------|---------------------|
| POST-DEP-001 | Monthly depreciation calculated | SUM(MonthlyDepreciation) = AnnualDepreciation / 12 | Reconciliation |
| POST-DEP-002 | Accumulated depreciation updated | AccumulatedDepreciation = SUM(AllPeriodDepreciation) | Query verification |
| POST-DEP-003 | Net book value accurate | NBV = AcquisitionCost - AccumulatedDepreciation | Calculation verification |

**Post-condition Verification:**

- Verify depreciation schedule totals equal expected annual depreciation
- Confirm net book value never goes below residual value

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Asset Placed in Service
  │
  ▼
Determine Depreciation Method (US GAAP vs IFRS)
  │
  ├─► US GAAP ─► Straight-Line / Double-Declining / Units of Production
  │
  └─► IFRS ─► Cost Model / Revaluation Model
        │
        ▼
    Calculate Depreciable Amount = Cost - Residual Value
        │
        ▼
    Apply Depreciation Method
        │
        ▼
    Generate Depreciation Schedule
        │
        ▼
    Post Monthly Journal Entries
        │
        ▼
    END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-DEP-001 | Straight-Line Formula | Method = STRAIGHT_LINE | Annual = (Cost - Salvage) / UsefulLife | 1 |
| BR-DEP-002 | Double-Declining Formula | Method = DOUBLE_DECLINING | Rate = (2 / UsefulLife), Annual = BookValue × Rate | 1 |
| BR-DEP-003 | Sum-of-Years Formula | Method = SUM_OF_YEARS | Fraction = (RemainingLife / SumOfYears) | 1 |
| BR-DEP-004 | Units of Production | Method = UNITS_OF_PRODUCTION | Rate = (Cost - Salvage) / TotalUnits | 1 |
| BR-DEP-005 | Half-Year Convention | US GAAP, no mid-quarter | First year = 50% of annual | 1 |
| BR-DEP-006 | Mid-Quarter Convention | US GAAP, mid-quarter election | Based on quarter placed in service | 2 |
| BR-DEP-007 | Depreciation Suspension | Asset.Status = 'IDLE' > 30 days | Suspend depreciation | 1 |
| BR-DEP-008 | Impairment Adjustment | ImpairmentIndicator = TRUE | Recalculate remaining depreciation | 1 |
| BR-DEP-009 | Residual Value Floor | NBV < ResidualValue | Stop depreciation at residual value | 1 |
| BR-DEP-010 | Mid-Month Convention | IFRS, some jurisdictions | Pro-rata from mid-month | 2 |

**Calculation Logic:**

```
STRAIGHT-LINE:
  AnnualDepreciation = (AcquisitionCost - ResidualValue) / UsefulLife
  MonthlyDepreciation = AnnualDepreciation / 12

DOUBLE-DECLINING BALANCE:
  DecliningRate = (2 / UsefulLife) × 100
  Year1Depreciation = AcquisitionCost × (DecliningRate / 100)
  Year2Depreciation = (Cost - Year1Depreciation) × (DecliningRate / 100)
  Switch to straight-line when SL > DDB

SUM-OF-YEARS DIGITS:
  SumOfYears = (UsefulLife × (UsefulLife + 1)) / 2
  YearNDepreciation = (Cost - ResidualValue) × ((UsefulLife - N + 1) / SumOfYears)

UNITS OF PRODUCTION:
  DepreciationPerUnit = (Cost - ResidualValue) / EstimatedTotalUnits
  PeriodDepreciation = DepreciationPerUnit × UnitsThisPeriod
```

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-DEP-001 | Asset with 0 residual value | Depreciate full cost | Standard calculation |
| EC-DEP-002 | Asset depreciated to below residual | Stop at residual value | BR-DEP-009 |
| EC-DEP-003 | Mid-year acquisition | Apply half-year convention | Pro-rata first period |
| EC-DEP-004 | Asset improvement added | Separate depreciation track | Create new depreciation book |
| EC-DEP-005 | Asset reclassified mid-life | Recalculate remaining | Trigger reclassification process |
| EC-DEP-006 | Zero useful life | Flag error | Reject with DEP-ERR-002 |
| EC-DEP-007 | Negative acquisition cost | Flag error | Reject with DEP-ERR-001 |
| EC-DEP-008 | Component replacement | Depreciate component separately | Component depreciation tracking |

**Edge Case Decision Matrix:**

| Condition | Condition | Condition | Outcome |
|-----------|-----------|-----------|---------|
| Cost > 0 | Residual <= Cost | Life >= 1 | Standard depreciation |
| Cost > 0 | Residual = 0 | Life >= 1 | Full cost depreciation |
| Cost > 0 | Residual < Cost | Life = 0 | Reject - error |
| Cost = 0 | Any | Any | Reject - error |
| Cost < 0 | Any | Any | Reject - error |

### 3.5 State Transitions

**State Diagram:**

```
States: DRAFT, PENDING_APPROVAL, ACTIVE, DEPRECIATING, SUSPENDED, IMPAIRED, DISPOSED, FULLY_DEPRECIATED

Transitions:
┌─────────────┐       ┌─────────────┐       ┌─────────────┐
│   DRAFT     │──────►│  ACTIVE     │──────►│DEPRECIATING │
└─────────────┘       └─────────────┘       └─────────────┘
       │                     │                     │
       │                     ▼                     ▼
       │              ┌─────────────┐       ┌─────────────┐
       └─────────────►│  DISPOSED  │◄──────│  SUSPENDED  │
                      └─────────────┘       └─────────────┘
                                              │
                                              ▼
                                       ┌─────────────┐
                                       │  IMPAIRED   │
                                       └─────────────┘
                                              │
                                              ▼
                                       ┌───────────────────┐
                                       │ FULLY_DEPRECIATED │
                                       └───────────────────┘
```

**Transition Rules:**

| Current State | Event | Next State | Guard Conditions | Side Effects |
|---------------|-------|------------|------------------|---------------|
| ACTIVE | First depreciation posted | DEPRECIATING | All pre-conditions met | Journal entry created |
| DEPRECIATING | Asset idled > 30 days | SUSPENDED | Idle flag set | Depreciation suspended |
| SUSPENDED | Asset returned to service | DEPRECIATING | Return date recorded | Resume depreciation |
| DEPRECIATING | Impairment indicator | IMPAIRED | Impairment assessment | Recalculate schedule |
| IMPAIRED | Continue depreciating | DEPRECIATING | New schedule applied | Updated schedule |
| DEPRECIATING | NBV = Residual Value | FULLY_DEPRECIATED | Period end | Final journal entry |
| Any (except DISPOSED) | Disposal | DISPOSED | Disposal date valid | Gain/loss calculated |
