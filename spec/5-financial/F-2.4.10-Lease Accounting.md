| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|

| PRE-LEASE-002 | Discount rate required | DiscountRate > 0 | LEASE-ERR-002 | "Discount rate must be positive" |
| PRE-LEASE-003 | Lease term defined | LeaseTerm > 0 | LEASE-ERR-003 | "Lease term must be greater than 0" |
| PRE-LEASE-004 | Payment schedule complete | SUM(Payments) > 0 | LEASE-ERR-004 | "Lease payments must be greater than 0" |
| PRE-LEASE-005 | Lessor information | LessorID IN ValidLessors | LEASE-ERR-005 | "Invalid lessor" |

**Pre-condition Validation Matrix:**

```
IF (Lease.LeaseType IN ('FINANCE', 'OPERATING'))
   AND (Lease.DiscountRate > 0)
   AND (Lease.LeaseTerm > 0)
   AND (Lease.PaymentCount > 0)
THEN calculate lease liability
ELSE throw LEASE-ERR-00X Error
```

|-------------------|-------------|-------------------|---------------------|

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Lease Contract Analysis
  │
  ▼
Identify Lease Classification
  │
  ├─► Finance Lease (ASC 842/IFRS 16)
  │     │识别 (Ownership transfer, BPO, PV > 90%, 75% term)
  │     ▼
  │ Calculate Right-of-Use Asset
  │     │Initial = PV(Payments) + InitialCosts - Incentives
  │     ▼
  │ Calculate Lease Liability
  │     │PV of future payments at discount rate
  │     ▼
  │ Post Interest + Depreciation
  │     │Interest = Liability × Rate
  │     │Depreciation = Asset / Term (straight-line)
  │
  └─► Operating Lease (ASC 842/IFRS 16)
        │Not meeting finance lease criteria
        ▼
        Calculate ROU Asset
        │Straight-line expense (ASC 842) OR
        │Asset + Liability (IFRS 16 - front-loaded)
        ▼
        Post Periodic Expense
              │
              ▼
          END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-LEASE-001 | Finance Lease Test | ASC 842/IFRS 16 criteria | Classify accordingly | 1 |
| BR-LEASE-002 | PV Calculation | Standard | PV = Σ(Payment / (1+r)^t) | 1 |
| BR-LEASE-003 | ROU Asset Initial | Finance lease | = Lease Liability + Initial Costs - Incentives | 1 |
| BR-LEASE-004 | Lease Liability Initial | Finance lease | = PV of Lease Payments | 1 |
| BR-LEASE-005 | Interest Expense | Finance lease | = Beginning Liability × Discount Rate | 1 |
| BR-LEASE-006 | ROU Depreciation | Finance lease | = ROU Asset / Lease Term (straight-line) | 1 |
| BR-LEASE-007 | Operating Lease Expense | ASC 842 | Straight-line expense | 1 |
| BR-LEASE-008 | Variable Payment | When incurred | Excluded from liability | 1 |
| BR-LEASE-009 | Short-term Exemption | LeaseTerm < 12 months | Expense as incurred | 1 |
| BR-LEASE-010 | Low-value Exemption | Asset Value < $5,000 | Expense as incurred | 1 |
| BR-LEASE-011 | Renewal Option Assessment | IFRS 16 | Reasonable certainty test | 1 |

**Calculation Logic:**

```
Finance Lease:
  Lease Liability = Σ(t=1 to n) [ LeasePayment_t / (1 + IBR)^t ]
  ROU Asset = Lease Liability + InitialDirectCosts - LeaseIncentives + dismantling costs
  
  Monthly Interest = Beginning Lease Liability × (IBR / 12)
  Monthly Depreciation = ROU Asset / Lease Term in months
  
Operating Lease (ASC 842):
  Periodic Expense = Straight-line of all payments
  
Operating Lease (IFRS 16):
  ROU Asset = Σ(t=1 to n) [ Payment_t / (1 + IBR)^t ] (front-loaded)
  Periodic Expense = Depreciation of ROU Asset + Interest

Lease Term = Non-cancellable Period + Renewal Options (if reasonably certain)
```

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-LEASE-001 | Variable lease payments | Exclude from liability | Record when incurred |
| EC-LEASE-002 | Lease modification | Remeasure liability | Modified payments |
| EC-LEASE-003 | Sublease | Track head lease and sublease | Net presentation |
| EC-LEASE-004 | Purchase option exercised | Transfer to owned asset | Reclassify asset |
| EC-LEASE-005 | Terminate early | Calculate gain/loss | Derecognize ROU and liability |
| EC-LEASE-006 | Asset impairment | Test ROU for impairment | IAS 36/ASC 360 |

**Edge Case Decision Matrix:**

| Condition | Condition | Condition | Outcome |
|-----------|-----------|-----------|---------|
| Ownership transfer | Yes | Any | Finance lease |
| BPO | Yes | Any | Finance lease |
| PV >= 90% Fair Value | Yes | Any | Finance lease |
| Term >= 75% Life | Yes | Any | Finance lease |
| None of above | No | IFRS 16 | Operating lease |
| None of above | No | ASC 842 | Operating lease |
