# F-2.4.4 - TCO Analysis

**Requirement ID:** F-2.4.4  
**Module:** Financial Management  
**Priority:** Medium

---

## Requirement Description

Calculate and analyze total cost of ownership for assets to support procurement decisions and lifecycle management. Includes acquisition cost, operating costs, maintenance costs, and disposal costs over asset lifetime with NPV calculation and sensitivity analysis.

---

## Business Rules

1. NPV calculation: Sum of discounted cash flows over useful life
2. Sensitivity analysis for key variables (±10%, ±20% scenarios)
3. Cost components: acquisition, operating, maintenance, and disposal
4. Discount rate must be between 0% and 30%
5. Inflation adjustment available for future cost projections
6. Alternative comparison by NPV ranking

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Acquisition Cost | Yes | Purchase price plus installation |
| Useful Life | Yes | Expected lifespan in years |
| Discount Rate | Yes | Rate for NPV calculation |
| Operating Cost (Annual) | Yes | Yearly operating expenses |
| Maintenance Cost (Annual) | Yes | Yearly maintenance expenses |
| Residual Value | Yes | Expected disposal value |

---

## Output Data

| Data | Description |
|------|-------------|
| Total TCO | Sum of all lifetime costs |
| NPV | Net present value of ownership |
| Annual Cost Average | Typical yearly cost |
| Sensitivity Analysis | Cost variations under different scenarios |

---

## Validation Rules

1. Acquisition cost must be positive
2. Useful life must be between 1 and 50 years
3. Discount rate must be between 0% and 30%
4. Residual value must be non-negative and less than acquisition cost
5. All cost components must be provided
6. Cost data source must be documented

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full access |

- **Data Level**: Company level
