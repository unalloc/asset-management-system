## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-TCO-001 | Acquisition cost must be positive | AcquisitionCost > 0 | TCO-ERR-001 | "Acquisition cost must be positive" |
| PRE-TCO-002 | Useful life must be reasonable | UsefulLife >= 1 AND UsefulLife <= 50 | TCO-ERR-002 | "Useful life must be between 1 and 50 years" |
| PRE-TCO-003 | Discount rate must be valid | DiscountRate >= 0 AND DiscountRate <= 0.30 | TCO-ERR-003 | "Discount rate must be between 0% and 30%" |
| PRE-TCO-004 | Cost data complete | All required cost components provided | TCO-ERR-004 | "All cost components are required" |
| PRE-TCO-005 | Residual value reasonable | ResidualValue >= 0 AND ResidualValue < AcquisitionCost | TCO-ERR-005 | "Residual value must be non-negative and less than acquisition cost" |

**Pre-condition Validation Matrix:**

```
IF (TCO.AcquisitionCost > 0)
   AND (TCO.UsefulLife >= 1 AND TCO.UsefulLife <= 50)
   AND (TCO.DiscountRate >= 0 AND TCO.DiscountRate <= 0.30)
   AND (TCO.OperatingCost IS NOT NULL)
   AND (TCO.MaintenanceCost IS NOT NULL)
THEN proceed to TCO calculation
ELSE throw TCO-ERR-00X Error
```

### 3.2 Post-conditions

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Define TCO Model Parameters
  │
  ├─► Acquisition Costs
  │     - Purchase price
  │     - Installation
  │     - Training
  │     - Transportation
  │
  ├─► Operating Costs (Annual)
  │     - Energy consumption
  │     - Consumables
  │     - Labor
  │     - Insurance
  │
  ├─► Maintenance Costs (Annual)
  │     - Preventive
  │     - Corrective
  │     - Spare parts
  │
  └─► Disposal/Residual
        - Disposal costs
        - Residual value
        │
        ▼
    Calculate Annual Cash Flows
        │
        ▼
    Apply Discount Rate for NPV
        │
        ▼
    Generate Sensitivity Analysis
        │
        ▼
    Compare Alternatives
        │
        ▼
    END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-TCO-001 | NPV Calculation | Standard | NPV = Σ(CFt / (1+r)^t) | 1 |
| BR-TCO-002 | Annual Cost Aggregation | Per year | TotalAnnualCost = Operating + Maintenance | 1 |
| BR-TCO-003 | Residual Value Treatment | Disposal | Add residual, subtract disposal cost | 1 |
| BR-TCO-004 | Sensitivity Variable | Each parameter | Calculate TCO for ±10%, ±20% | 1 |
| BR-TCO-005 | Inflation Adjustment | InflationRate > 0 | Adjust future costs | 2 |
| BR-TCO-006 | Alternative Comparison | Multiple options | Rank by NPV | 1 |
| BR-TCO-007 | Break-even Analysis | Single option | Find year where cumulative cost = benefit | 2 |

**Calculation Logic:**

```
TCO = AcquisitionCost + Σ(AnnualOperatingCost + AnnualMaintenanceCost) - ResidualValue + DisposalCost

NPV = Σ(t=1 to n) [ (OperatingCost_t + MaintenanceCost_t) / (1 + r)^t ] + AcquisitionCost / (1+r)^0 - ResidualValue / (1+r)^n

Where:
  t = year
  r = discount rate
  n = useful life

Sensitivity Analysis:
  Base Case: All parameters at expected values
  Best Case: All parameters at optimistic values
  Worst Case: All parameters at pessimistic values
  One-Way Sensitivity: Each parameter varies while others held constant
```

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-TCO-001 | Zero maintenance costs | Valid scenario | Allow with documentation |
| EC-TCO-002 | Negative residual value (liability) | Treat as disposal cost | Add to final year cost |
| EC-TCO-003 | Discount rate = 0 | NPV = simple sum | Allow calculation |
| EC-TCO-004 | Infinite useful life (land) | Flag as non-depreciable | Separate treatment |
| EC-TCO-005 | Missing cost data | Require completion | Block with error |
| EC-TCO-006 | Negative NPV | Signal investment loss | Document and flag |

**Edge Case Decision Matrix:**

| Condition | Condition | Condition | Outcome |
|-----------|-----------|-----------|---------|
| DiscountRate >= 0 | DiscountRate <= 0.30 | UsefulLife > 0 | Valid calculation |
| DiscountRate < 0 | Any | Any | Reject - invalid rate |
| DiscountRate > 0.30 | Any | Any | Warn - high rate |
| UsefulLife = 0 | Any | Any | Reject - no life |
| UsefulLife > 50 | Any | Any | Warn - verify |
