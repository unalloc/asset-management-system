### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Identify Reclassification Need
  │
  ├─► Impairment Trigger Detected
  │     │
  │     ▼
  │ Calculate Recoverable Amount
  │     │
  │     ├─► Fair Value - Costs to Sell
  │     └─► Value in Use (DCF)
  │           │
  │           ▼
  │     Compare to Carrying Amount
  │           │
  │           ├─► Recoverable < Carrying ─► Record Impairment
  │           └─► Recoverable >= Carrying ─► No impairment
  │
  └─► Classification Change
        │
        ▼
    Evaluate New Classification
        │
        ├─► CIP to Fixed Asset ─► Capitalize
        ├─► Fixed Asset to Investment ─► Apply IAS 40/ASC 350
        ├─► Category Change ─► Update depreciation
        └─► Company Transfer ─► Handle intercompany
              │
              ▼
          Apply Change
              │
              ▼
          END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-RECLASS-001 | Recoverable Amount | Impairment test | MAX(FairValue - CostsToSell, ValueInUse) | 1 |
| BR-RECLASS-002 | Value in Use Calculation | DCF approach | Σ(CFt / (1+r)^t) for remaining life | 1 |
| BR-RECLASS-003 | Impairment Loss | Recoverable < Carrying | Impairment = Carrying - Recoverable | 1 |
| BR-RECLASS-004 | Impairment Reversal | IAS 36, not goodwill | Allow reversal up to original loss | 2 |
| BR-RECLASS-005 | CIP Capitalization | Substantial completion | Transfer to fixed asset | 1 |
| BR-RECLASS-006 | Investment Property | IAS 40 qualified | Apply fair value model | 2 |

**Calculation Logic:**

```
Recoverable Amount = MAX(Fair Value - Costs to Sell, Value in Use)

Value in Use = Σ(t=1 to n) [ CFt / (1 + WACC)^t ]

Impairment Loss = MAX(0, Carrying Amount - Recoverable Amount)

Where:
  CFt = Cash flow in year t
  WACC = Weighted average cost of capital
  n = Remaining useful life

Post-impairment carrying amount = Carrying Amount - Impairment Loss
```

### 3.4 Edge Cases

| Edge Case ID | Scenario | Expected Behavior | Handling Approach |
|--------------|----------|-------------------|-------------------|
| EC-RECLASS-001 | Reversal of previous impairment | Allow per IAS 36 | Calculate reversal limit |
| EC-RECLASS-002 | Goodwill impairment | Non-reversible | Track separately |
| EC-RECLASS-003 | CGU impairment | Allocate to assets | Pro-rata allocation |
| EC-RECLASS-004 | Investment property reclassification | Apply prospectively | IAS 40 transition rules |
| EC-RECLASS-005 | Asset under construction | Defer impairment test | Test when complete |
| EC-RECLASS-006 | Recoverable amount = 0 | Full impairment | Write off asset |

**Edge Case Decision Matrix:**

| Condition | Condition | Condition | Outcome |
|-----------|-----------|-----------|---------|
| Recoverable > Carrying | No indicator | Any | No impairment |
| Recoverable < Carrying | First impairment | Any | Record loss |
| Recoverable < Carrying | Subsequent | IAS 36 asset | Allow reversal |
| Recoverable < Carrying | Subsequent | Goodwill | No reversal |
