## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-TAX-001 | Tax jurisdiction must be valid | JurisdictionCode IN ValidJurisdictions | TAX-ERR-001 | "Invalid tax jurisdiction" |
| PRE-TAX-002 | Tax depreciation policy configured | TaxBook exists for jurisdiction | TAX-ERR-002 | "Tax depreciation policy not configured" |
| PRE-TAX-003 | Asset classification for tax | TaxCategory assigned | TAX-ERR-003 | "Asset must have tax classification" |
| PRE-TAX-004 | Filing period valid | TaxPeriod IS WITHIN ValidFilingPeriod | TAX-ERR-004 | "Tax period is outside valid filing window" |

**Pre-condition Validation Matrix:**

```
IF (Asset.TaxJurisdiction IS NOT NULL)
   AND (Asset.TaxCategory IS NOT NULL)
   AND (TaxBook EXISTS FOR jurisdiction)
   AND (TaxPeriod IS VALID)
THEN proceed to tax reporting
ELSE throw TAX-ERR-00X Error
```

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Identify Tax Jurisdiction
  │
  ▼
Determine Tax Classification
  │
  ├─► US ─► Apply MACRS/ACRS Tables
  ├─► Germany ─► Apply AfA Tables
  ├─► France ─► Apply Amortissement Tables
  └─► Other ─► Apply Country-Specific Rules
        │
        ▼
    Calculate Tax Depreciation
        │
        ▼
    Calculate Book-Tax Differences
        │
        ▼
    Generate Tax Forms
        │
        ▼
    Apply Tax Credits/Incentives
        │
        ▼
    END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-TAX-001 | MACRS 200% DB | US Tax, property class | Apply 200% declining balance | 1 |
| BR-TAX-002 | MACRS 150% DB | US Tax, alternative | Apply 150% declining balance | 2 |
| BR-TAX-003 | Straight-Line Alternative | US Tax, PSL | Apply straight-line if elected | 2 |
| BR-TAX-004 | Half-Year Convention | US Tax, standard | 50% first year | 1 |
| BR-TAX-005 | Mid-Quarter Convention | US Tax, mid-quarter | Based on Qtr placed in service | 1 |
| BR-TAX-006 | Section 179 Deduction | US Tax, eligible | Allow up to limit | 1 |
| BR-TAX-007 | Bonus Depreciation | US Tax, when enacted | Apply per legislation | 1 |
| BR-TAX-008 | German AfA | Germany | Apply table rates | 1 |
| BR-TAX-009 | Book-Tax Difference | All | Calculate temporary/permanent | 1 |

**Calculation Logic:**

```
US MACRS Depreciation:
  DepreciationRate = MACRSTable(PropertyClass, Method)
  TaxDepreciation = DepreciableBasis × DepreciationRate

Section 179 Deduction (when applicable):
  Section179Deduction = MIN(Cost, Section179Limit)
  RemainingBasis = Cost - Section179Deduction
  RemainingDepreciation = Depreciate(RemainingBasis)

Book-Tax Difference:
  TemporaryDiff = BookDepreciation - TaxDepreciation
  This difference reverses over asset life

Deferred Tax Liability:
  DTL = TemporaryDiff × TaxRate
```

**Cross-Field Validation Logic:**

```
IF (Asset.TaxJurisdiction = 'US')
   AND (Asset.TaxCategory = '179_PROPERTY')
   AND (Asset.TaxCostBasis > US_SECTION179_LIMIT)
THEN apply SECTION179_LIMIT
     AND display TAX-WRN-003 "Section 179 deduction limited to $1,160,000"

IF (TaxDepreciation.Total > TaxDepreciation.TaxCostBasis)
```
