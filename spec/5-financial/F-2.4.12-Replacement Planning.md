## 3. Business Rules Enhancement

### 3.1 Pre-conditions

| Pre-condition ID | Description | Validation Logic | Error Code | Error Message |
|------------------|-------------|------------------|------------|---------------|
| PRE-REPL-001 | Asset status available | Asset.Status IN ('ACTIVE', 'DEPRECIATING') | REPL-ERR-001 | "Asset must be active to plan replacement" |
| PRE-REPL-002 | Remaining useful life calculated | RemainingLife >= 0 | REPL-ERR-002 | "Remaining life must be non-negative" |
| PRE-REPL-003 | Cost data available | CostHistory exists | REPL-ERR-003 | "Cost history required for replacement analysis" |
| PRE-REPL-004 | Replacement cost estimated | ReplacementCost > 0 | REPL-ERR-004 | "Replacement cost must be positive" |

**Pre-condition Validation Matrix:**

```
IF (Asset.Status IN ('ACTIVE', 'DEPRECIATING'))
   AND (Asset.RemainingLife IS NOT NULL)
   AND (CostHistory EXISTS FOR Asset)
   AND (ReplacementCost > 0)
THEN generate replacement analysis
ELSE throw REPL-ERR-00X Error
```

### 3.3 Business Logic

**Decision Tree:**

```
START
  │
  ▼
Identify Replacement Candidates
  │
  ├─► Age-based (useful life exceeded)
  ├─► Condition-based (degradation detected)
  ├─► Performance-based (below threshold)
  └─► Risk-based (failure probability high)
        │
        ▼
    Calculate Replacement Factors
      │
      ├─► Remaining useful life
      ├─► Maintenance cost trend
      ├─► Failure probability
      ├─► Downtime cost impact
      └─► Technology obsolescence
            │
            ▼
    Generate Priority Score
      │
      ▼
    Compare: Keep vs Replace
      │
      ├─► NPV of keeping > NPV of replacing ─► Recommend Keep
      └─► NPV of replacing > NPV of keeping ─► Recommend Replace
            │
            ▼
        Create Replacement Plan
          │
          ▼
        END
```

**Business Rules Specification:**

| Rule ID | Rule Name | Condition | Action | Priority |
|---------|-----------|-----------|--------|----------|
| BR-REPL-001 | Priority Score Calculation | Standard | Weighted sum of factors | 1 |
| BR-REPL-002 | Replacement NPV | New asset costs | NPV = Cost - Salvage + Operating | 1 |
| BR-REPL-003 | Keep NPV | Current asset costs | NPV = Maintenance + Downtime | 1 |
| BR-REPL-004 | Optimal Timing | Min life-cycle cost | Find year with lowest EAC | 1 |
| BR-REPL-005 | Budget Integration | Capital planning | Align with available funds | 1 |
| BR-REPL-006 | Risk Adjustment | High-risk assets | Accelerate replacement | 1 |

**Calculation Logic:**

```
Priority Score = (AgeScore × 0.3) + (ConditionScore × 0.3) + (CostScore × 0.2) + (RiskScore × 0.2)

Equivalent Annual Cost (EAC):
  EAC = (PurchaseCost - SalvageValue) × [r × (1+r)^n] / [(1+r)^n - 1] + AnnualMaintenance

Replace if: EAC_New + DowntimeCost < EAC_Current + DowntimeRisk

Remaining Life = MAX(0, UsefulLife - AssetAge)
```
