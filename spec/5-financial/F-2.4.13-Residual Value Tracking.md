# F-2.4.13 - Residual Value Tracking

**Requirement ID:** F-2.4.13  
**Module:** Financial Management  
**Priority:** Medium

---

## Requirement Description

Track asset estimated and actual residual values to optimize disposal decisions. Includes initial residual value estimation, periodic market assessments, variance analysis, and gain/loss recording at disposal.

---

## Business Rules

1. Initial residual value set as percentage of cost based on asset category
2. Annual review recommended for all assets
3. Disposal gain/loss = proceeds - net book value
4. Realization rate = actual proceeds / estimated residual x 100%
5. Market assessment or justification required for residual value changes
6. Negative NBV at disposal treated as additional loss

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Asset | Yes | Asset with residual value |
| Estimated Residual Value | Yes | Expected value at end of life |
| Market Assessment | No | Evidence supporting the estimate |
| Assessor | Yes | Person conducting assessment |
| Assessment Date | Yes | When assessment was done |

---

## Output Data

| Data | Description |
|------|-------------|
| Residual Value Assessment | Market-based estimate |
| Disposal Record | Proceeds and gain/loss calculation |
| Variance Report | Estimated vs actual comparison |

---

## Validation Rules

1. Estimated value cannot be negative
2. Must be between 0 and acquisition cost
3. Market assessment or justification required
4. Assets near end of life flagged for review

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full access |
| Company Finance | Create assessments, record disposals |
| Plant Manager | View within plant |

---

## Multi-Company Data Scope

- **Data Level**: Company level with plant viewing
- **Access**: Assessments at company and division level
- **Isolation**: Company Code scopes all tracking |

---

## Compliance Requirements

- ASC 360 (US GAAP) gain/loss on disposal
- WEEE Directive disposal tracking for electronic assets
- SOX disposal approval and documentation
