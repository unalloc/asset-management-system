# F-2.4.7 - Capital Projects

**Requirement ID:** F-2.4.7  
**Module:** Financial Management  
**Priority:** Medium

---

## Requirement Description

Track Construction in Progress (CIP) projects and manage the complete process from project start to asset capitalization. Includes budget tracking, expenditure categorization, interest capitalization, and automated asset creation upon completion.

---

## Business Rules

1. Direct costs capitalized to asset
2. Indirect costs allocated by percentage
3. Interest capitalization for qualifying assets (weighted average accumulated expenditures x capitalization rate)
4. Overhead capitalization per policy
5. Asset created when project substantially complete
6. Non-qualifying costs expensed

---

## Input Data

| Data | Required | Description |
|------|----------|-------------|
| Project Code | Yes | Unique identifier for the CIP project |
| Project Name | Yes | Descriptive name |
| Budget Amount | Yes | Approved project budget |
| Start Date | Yes | When project begins |
| Estimated End Date | Yes | Expected completion |
| Capitalization Threshold | Yes | Minimum cost to capitalize |

---

## Output Data

| Data | Description |
|------|-------------|
| CIP Project Record | Project details and status |
| Expenditure Tracking | Categorized project costs |
| Capitalization Entry | Asset created from completed project |
| Cost Summary | Direct, indirect, interest, overhead |

---

## Validation Rules

1. Project code must be unique
2. Budget amount must be positive
3. Start date must be before end date
4. Budget must be approved before tracking
5. Over-budget requires approval
6. Total cost must meet capitalization threshold

---

## Access Control

| Role | Permission |
|------|------------|
| System Admin | Full access |
| Company Controller | Create projects, approve capitalization |
| Division Manager | View and approve within division |
| Plant Manager | View and track within plant |

---

## Multi-Company Data Scope

- **Data Level**: Company level with division scoping
- **Access**: Project creation at company and division level
- **Isolation**: Company Code scopes all CIP data |

---

## Compliance Requirements

- ASC 360 (US GAAP) capitalization criteria
- IAS 16 (IFRS) costs to include in asset cost
- ASC 835-20 / IAS 23 interest capitalization
- SOX 404 internal controls over capital projects
